```ts
function setLightingForKeyWithHidCode(
  keyCode: number,
  redPercentage: number,
  greenPercentage: number,
  bluePercentage: number,
  callback: CallbackFunction<Result>,
): void;
```

Sets the lighting for a specific key by HID code.

## Parameters

| Parameter         | Type                                                                                                                                            | Description                                |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------ |
| `keyCode`         | `number`                                                                                                                                        | The key HID code.                          |
| `redPercentage`   | `number`                                                                                                                                        | Red percentage (0 - 100)                   |
| `greenPercentage` | `number`                                                                                                                                        | Green percentage (0 - 100)                 |
| `bluePercentage`  | `number`                                                                                                                                        | Blue percentage (0 - 100)                  |
| `callback`        | [`CallbackFunction`](../../../../../../../../overwolf.io/type-aliases/CallbackFunction.mdx)\<[`Result`](../../../../../interfaces/Result.mdx)\> | A callback with the result of the request. |

## Returns

`void`
