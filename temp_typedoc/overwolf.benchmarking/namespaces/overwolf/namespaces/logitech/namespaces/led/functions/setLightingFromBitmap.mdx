## Call Signature

```ts
function setLightingFromBitmap(
  bitmapUrl: string,
  callback: CallbackFunction<Result>,
): void;
```

Sets the lighting from an overwolf-extension:// or overwolf-media:// url.
The file must be 21x6.

### Parameters

| Parameter   | Type                                                                                                                                            | Description                                |
| ----------- | ----------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------ |
| `bitmapUrl` | `string`                                                                                                                                        | The Overwolf url to add.                   |
| `callback`  | [`CallbackFunction`](../../../../../../../../overwolf.io/type-aliases/CallbackFunction.mdx)\<[`Result`](../../../../../interfaces/Result.mdx)\> | A callback with the result of the request. |

### Returns

`void`
