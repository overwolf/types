## Call Signature

```ts
function changeSize(
  windowId: string,
  width: number,
  height: number,
  callback?: CallbackFunction<Result>,
): void;
```

Changes the window size to the new width and height, in pixels.

### Parameters

| Parameter   | Type                                                                                                                                | Description                                                   |
| ----------- | ----------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------- |
| `windowId`  | `string`                                                                                                                            | The id or name of the window for which to change the size.    |
| `width`     | `number`                                                                                                                            | The new window width in pixels.                               |
| `height`    | `number`                                                                                                                            | The new window height in pixels.                              |
| `callback?` | [`CallbackFunction`](../../../../../../overwolf.io/type-aliases/CallbackFunction.mdx)\<[`Result`](../../../interfaces/Result.mdx)\> | A callback which is called when the size change is completed. |

### Returns

`void`

## Call Signature

```ts
function changeSize(
  changeSizeParams: ChangeWindowSizeParams,
```

### Parameters

| Parameter          | Type                                                                                                                                | Description                                                   |
