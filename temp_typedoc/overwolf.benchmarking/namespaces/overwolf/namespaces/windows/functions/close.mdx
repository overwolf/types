```ts
function close(
  windowId: string,
  callback?: CallbackFunction<WindowIdResult>,
): void;
```

Closes the window.

## Parameters

| Parameter   | Type                                                                                                                                          | Description                            |
| ----------- | --------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------- |
| `windowId`  | `string`                                                                                                                                      | The id or name of the window to close. |
| `callback?` | [`CallbackFunction`](../../../../../../overwolf.io/type-aliases/CallbackFunction.mdx)\<[`WindowIdResult`](../interfaces/WindowIdResult.mdx)\> | Called after the window is closed.     |

## Returns

`void`
