```ts
function getWindowState(
  windowId: string,
  callback: CallbackFunction<GetWindowStateResult>,
): void;
```

Returns the state of the window (normal/minimized/maximized/closed).

## Parameters

| Parameter  | Type                                                                                                                                                      | Description                   |
| ---------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------- |
| `windowId` | `string`                                                                                                                                                  | The id or name of the window. |
| `callback` | [`CallbackFunction`](../../../../../../overwolf.io/type-aliases/CallbackFunction.mdx)\<[`GetWindowStateResult`](../interfaces/GetWindowStateResult.mdx)\> | Called with the window state. |

## Returns

`void`
