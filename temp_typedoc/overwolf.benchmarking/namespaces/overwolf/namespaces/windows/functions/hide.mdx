```ts
function hide(
  windowId: string,
  callback?: CallbackFunction<WindowIdResult>,
): void;
```

Hides the window.

## Parameters

| Parameter   | Type                                                                                                                                          | Description                           |
| ----------- | --------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------- |
| `windowId`  | `string`                                                                                                                                      | The id or name of the window to hide. |
| `callback?` | [`CallbackFunction`](../../../../../../overwolf.io/type-aliases/CallbackFunction.mdx)\<[`WindowIdResult`](../interfaces/WindowIdResult.mdx)\> | Called after the window is hidden.    |

## Returns

`void`
