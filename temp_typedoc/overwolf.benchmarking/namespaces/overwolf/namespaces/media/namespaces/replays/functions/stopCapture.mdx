## Call Signature

```ts
function stopCapture(
  replayId: string,
  callback: CallbackFunction<ReplayResult>,
): void;
```

Finishes capturing a replay and returns a url to the created video file.
You can only call this method if replay mode is on and using a valid id of
a replay being captured to a file.

### Parameters

| Parameter  | Type                                                                                                                                            | Description                                                              |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------ |
| `replayId` | `string`                                                                                                                                        | The id of the replay you want to finish capturing.                       |
| `callback` | [`CallbackFunction`](../../../../../../../../overwolf.io/type-aliases/CallbackFunction.mdx)\<[`ReplayResult`](../interfaces/ReplayResult.mdx)\> | A callback function which will be called with the status of the request. |

### Returns

`void`
