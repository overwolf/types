## Functions

| Function                                                         | Description                                                                                                                                                                                                                                                                                                |
| ---------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [~~capture~~](functions/capture.mdx)                             | Starts capturing a replay to a file. A replay id will be returned in the callback which is needed to finish capturing the replay. You can only call this method if replay mode is on and no other replay is currently being captured to a file.                                                            |
| [~~getHighlightsFeatures~~](functions/getHighlightsFeatures.mdx) | Get supported auto highlights features for a game                                                                                                                                                                                                                                                          |
| [~~getState~~](functions/getState.mdx)                           | Returns whether replay capturing is turned on or off.                                                                                                                                                                                                                                                      |
| [~~setReplaysSubFolder~~](functions/setReplaysSubFolder.mdx)     | change target sub folder of current running replay provider                                                                                                                                                                                                                                                |
| [~~startCapture~~](functions/startCapture.mdx)                   | Starts capturing a replay to a file. A replay id will be returned in the callback which is needed to finish capturing the replay. You can only call this method if replay mode is on and no other replay is currently being captured to a file.                                                            |
| [~~stopCapture~~](functions/stopCapture.mdx)                     | Finishes capturing a replay and returns a url to the created video file. You can only call this method if replay mode is on and using a valid id of a replay being captured to a file.                                                                                                                     |
| [~~turnOff~~](functions/turnOff.mdx)                             | Turns off background replay capturing. Call this as soon as you no longer interesting in capturing, in order to free up resources.                                                                                                                                                                         |
| [~~turnOn~~](functions/turnOn.mdx)                               | Turns on background replay capturing. Without calling it first, you will not be able to create video replays. Notice that turning on replay capturing will consume system resources so use it wisely.buffer_length defines the amount of time in milliseconds to have captured in the memory at all times. |
