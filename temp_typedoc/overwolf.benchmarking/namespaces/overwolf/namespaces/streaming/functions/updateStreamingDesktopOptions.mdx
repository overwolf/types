Update stream desktop capture options.

## Parameters

| Parameter                    | Type                                                                                                                                | Description                                                     |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------- |
| `streamId`                   | `number`                                                                                                                            | The id of the stream for which to set the Be Right Back image.  |
| `newOptions`                 | [`StreamDesktopCaptureOptions`](../interfaces/StreamDesktopCaptureOptions.mdx)                                                      | The updated desktop capture streaming options.                  |
| `mouseCursorStreamingMethod` | [`StreamMouseCursor`](../namespaces/enums/enumerations/StreamMouseCursor.mdx)                                                       | The updated value of the mouse cursor streaming method.         |
| `callback?`                  | [`CallbackFunction`](../../../../../../overwolf.io/type-aliases/CallbackFunction.mdx)\<[`Result`](../../../interfaces/Result.mdx)\> | A callback function to call with success or failure indication. |

## Returns

`void`
