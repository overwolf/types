Stream desktop capture options.

## Properties

| Property                                                 | Type      | Description                                                                  |
| -------------------------------------------------------- | --------- | ---------------------------------------------------------------------------- |
| <a id="property-enable"></a> ~~`enable`~~                | `boolean` | Defines if to capture the desktop while game is not running or not in focus. |
| <a id="property-force_capture"></a> ~~`force_capture?`~~ | `boolean` | Defines if to force desktop streaming even when a game is in foreground.     |
| <a id="property-monitor_id"></a> ~~`monitor_id?`~~       | `number`  | Defines which monitor to stream when streaming desktop.                      |
