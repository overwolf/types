[Overwolf APIs](../../../../../../../../Overview.mdx) / [overwolf.benchmarking](../../../../../../../Overview.mdx) / [overwolf](../../../../../Overview.mdx) / [social](../../../Overview.mdx) / [youtube](../Overview.mdx) / share

```ts
function share(
  youTubeShareParams: ShareParameters,
  callback: CallbackFunction<Result>,
): void;
```

If the user is currently logged into YouTube, this will perform the video
share.

Possible errors that can occur:

- Disconnected (user isn't signed in)
- MissingFile (trying to share a missing file)
- UnsupportedFile (trying to share an unsupported format)

## Parameters

| Parameter            | Type                                                                                                                                            | Description                             |
| -------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------- |
| `youTubeShareParams` | [`ShareParameters`](../interfaces/ShareParameters.mdx)                                                                                          | The share parameters.                   |
| `callback`           | [`CallbackFunction`](../../../../../../../../overwolf.io/type-aliases/CallbackFunction.mdx)\<[`Result`](../../../../../interfaces/Result.mdx)\> | Will contain the status of the request. |

## Returns

`void`
