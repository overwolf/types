```ts
function fileExists(
  filePath: string,
  callback: CallbackFunction<FileExistsResult>,
```

Checks for the existence of the file in the given path.

## Parameters

| Parameter  | Type                                                                                                                                              | Description              |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------ |
| `filePath` | `string`                                                                                                                                          | The path to check for.   |
| `callback` | [`CallbackFunction`](../../../../../../overwolf.io/type-aliases/CallbackFunction.mdx)\<[`FileExistsResult`](../interfaces/FileExistsResult.mdx)\> | Returns with the result. |
