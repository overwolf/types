function sendLogs(

| Parameter     | Type                                                                                                                                | Description                                |

| `callback`    | [`CallbackFunction`](../../../../../../overwolf.io/type-aliases/CallbackFunction.mdx)\<[`Result`](../../../interfaces/Result.mdx)\> | A callback with the status of the request. |
