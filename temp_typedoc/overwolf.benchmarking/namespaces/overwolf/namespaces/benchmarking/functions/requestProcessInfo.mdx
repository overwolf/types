```ts
function requestProcessInfo(
  interval: number,
  callback: CallbackFunction<Result>,
): void;
```

Requests process information within a given interval. See
`requestPermissions` for administrative permissions instructions.

## Parameters

| Parameter  | Type                                                                                                                                | Description                                                                                         |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------- |
| `interval` | `number`                                                                                                                            | The desired maximal interval (in milliseconds) in which events will be triggered. Minimum is 500ms. |
| `callback` | [`CallbackFunction`](../../../../../../overwolf.io/type-aliases/CallbackFunction.mdx)\<[`Result`](../../../interfaces/Result.mdx)\> | A callback function which will be called with the status of the request.                            |

## Returns

`void`
