Defines the configuration for the NVIDIA NVENC encoder.

## Properties

| Property                                                        | Type                                                                                                      | Description                                                   |
| --------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------- |
| <a id="property-keyframe_interval"></a> ~~`keyframe_interval`~~ | `number`                                                                                                  | Defines the time, in seconds, after which to send a keyframe. |
| <a id="property-preset"></a> ~~`preset?`~~                      | `StreamEncoderPreset_NVIDIA`           | Defines which preset the encoder should use.                  |
| <a id="property-rate_control"></a> ~~`rate_control?`~~          | `StreamEncoderRateControl_NVIDIA` | Defines the rate control mode the encoder should use.         |
