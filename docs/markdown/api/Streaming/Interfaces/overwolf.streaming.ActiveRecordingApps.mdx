## Properties

| Property                                        | Type     | Description                            |
| ----------------------------------------------- | -------- | -------------------------------------- |
| <a id="property-displayname"></a> `displayName` | `string` | The display name of the recording app. |
| <a id="property-uid"></a> `uid`                 | `string` | The unique ID of the recording app.    |
