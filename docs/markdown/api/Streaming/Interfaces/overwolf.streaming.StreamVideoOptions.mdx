Overwolf APIs / overwolf / overwolf / streaming / StreamVideoOptions

Stream video options.

## Properties

| Property                                                                                       | Type                                                                                          | Description                                                                                                                                                                                              |
| ---------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <a id="property-auto_calc_kbps"></a> `auto_calc_kbps?`                                         | `boolean`                                                                                     | Defines if to try to automatically calculate the kbps. If set to true, then the max_kbps field is ignored.                                                                                               |
| <a id="property-base_frame_size_source"></a> `base_frame_size_source?`                         | `eVideoBaseFrameSizeSource` | -                                                                                                                                                                                                        |
| <a id="property-buffer_length"></a> `buffer_length?`                                           | `number`                                                                                      | Defines the length of the buffer to be recorded in millisenconds (max 40 seconds)                                                                                                                        |
| <a id="property-capture_desktop"></a> `capture_desktop?`                                       | `StreamDesktopCaptureOptions`                              | Defines the desktop streaming options.                                                                                                                                                                   |
| <a id="property-disable_auto_shutdown_on_game_exit"></a> `disable_auto_shutdown_on_game_exit?` | `boolean`                                                                                     | Disables automatic shutdown of the streaming API once the targeted game session ended.                                                                                                                   |
| <a id="property-disable_when_sht_not_supported"></a> `disable_when_sht_not_supported?`         | `boolean`                                                                                     | Do not start video replay service in case shared texture is not supported.                                                                                                                               |
| <a id="property-enable_on_demand_split"></a> `enable_on_demand_split?`                         | `boolean`                                                                                     | -                                                                                                                                                                                                        |
| <a id="property-encoder"></a> `encoder?`                                                       | `StreamingVideoEncoderSettings`                          | Defines the video encoder settings to use.                                                                                                                                                               |
| <a id="property-fps"></a> `fps?`                                                               | `number`                                                                                      | Defines the Frames Per Second for the stream.                                                                                                                                                            |
| <a id="property-frame_size_method"></a> `frame_size_method?`                                   | `eVideoFrameSizeCalcMethod` | -                                                                                                                                                                                                        |
| <a id="property-game_window_capture"></a> `game_window_capture?`                               | `GameWindowCapture`                                                  | -                                                                                                                                                                                                        |
| <a id="property-height"></a> `height?`                                                         | `number`                                                                                      | Defines the stream height in pixels.                                                                                                                                                                     |
| <a id="property-include_full_size_video"></a> `include_full_size_video?`                       | `boolean`                                                                                     | In case `max_file_size_bytes` is on, full video will be recorded to disk, parallel to splits videos.                                                                                                     |
| <a id="property-indication_position"></a> `indication_position?`                               | `IndicationPosition`               | Position of the recorder indicator. Available for video capture only.                                                                                                                                    |
| <a id="property-indication_type"></a> `indication_type?`                                       | `IndicationType`                       | Type of the recorder indicator. Available for video capture only.                                                                                                                                        |
| <a id="property-keep_game_capture_on_lost_focus"></a> `keep_game_capture_on_lost_focus?`       | `boolean`                                                                                     | Keep capturing the game when the game loses focus (i.e do not show "Be Right Back"). Note: if game is minimized, BRB will be shown.                                                                      |
| <a id="property-max_file_size_bytes"></a> `max_file_size_bytes?`                               | `number`                                                                                      | Defines file maximum size. when video reach `max_file_size_bytes`, the recorder will flash the video file and stat a new video file. `onFileSpilt` event will be fired.                                  |
| <a id="property-max_kbps"></a> `max_kbps?`                                                     | `number`                                                                                      | Defines the maximum KB per second of the stream.                                                                                                                                                         |
| <a id="property-notify_dropped_frames_ratio"></a> `notify_dropped_frames_ratio?`               | `number`                                                                                      | The ratio of dropped to non-dropped frames for which to issue a notification.                                                                                                                            |
| <a id="property-override_overwolf_setting"></a> `override_overwolf_setting?`                   | `boolean`                                                                                     | Do not use Overwolf capture setting. In case True you must provider all video setting (encoder..)                                                                                                        |
| <a id="property-sources"></a> `sources?`                                                       | `VideoSource`[]                  | Add sources to video (currently only webcam is supported)                                                                                                                                                |
| <a id="property-sub_folder_name"></a> `sub_folder_name?`                                       | `string`                                                                                      | Defines Sub folder for video file path destination (Optional). OverwolfVideoFolder\AppName\<sub_folder_name>\<file_name> In case `folder_name` is empty: `OverwolfVideoFolder\AppName\<sub_folder_name>` |
| <a id="property-test_drop_frames_interval"></a> `test_drop_frames_interval?`                   | `number`                                                                                      | The interval, in milliseconds, in which to test for dropped frames.                                                                                                                                      |
| <a id="property-use_app_display_name"></a> `use_app_display_name?`                             | `boolean`                                                                                     | use the app "short name" as the folder name, instead of using the app name from the manifest.                                                                                                            |
| <a id="property-width"></a> `width?`                                                           | `number`                                                                                      | Defines the stream width in pixels.                                                                                                                                                                      |
