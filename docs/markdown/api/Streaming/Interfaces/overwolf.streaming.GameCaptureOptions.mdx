Overwolf APIs / overwolf / overwolf / streaming / GameCaptureOptions

Options for filtering which game processes have their audio captured.

## Properties

| Property                                                                  | Type       | Description                                                                          |
| ------------------------------------------------------------------------- | ---------- | ------------------------------------------------------------------------------------ |
| <a id="property-additional_process_names"></a> `additional_process_names` | `string`[] | Additional process names (beyond the main game process) to include in audio capture. |
| <a id="property-enable"></a> `enable`                                     | `boolean`  | Whether filtered game audio capture is enabled.                                      |
