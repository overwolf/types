Overwolf APIs / overwolf / overwolf / streaming / StreamingVideoEncoderAMD_AMFSettings

Defines the configuration for the AMD AMF encoder.

## Properties

| Property                                                    | Type                                                                                                        | Description                                                   |
| ----------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------- |
| <a id="property-keyframe_interval"></a> `keyframe_interval` | `number`                                                                                                    | Defines the time, in seconds, after which to send a keyframe. |
| <a id="property-preset"></a> `preset?`                      | `StreamEncoderPreset_AMD_AMF`           | Defines which preset the encoder should use.                  |
| <a id="property-rate_control"></a> `rate_control?`          | `StreamEncoderRateControl_AMD_AMF` | Defines the rate control mode the encoder should use.         |
