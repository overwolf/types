Overwolf APIs / overwolf / overwolf / streaming / AudioDeviceData

Metadata describing an available audio device.

## Properties

| Property                                                    | Type      | Description                                              |
| ----------------------------------------------------------- | --------- | -------------------------------------------------------- |
| <a id="property-can_playback"></a> `can_playback`           | `boolean` | Whether the device supports audio playback (output).     |
| <a id="property-can_record"></a> `can_record`               | `boolean` | Whether the device supports audio recording (input).     |
| <a id="property-device_id"></a> `device_id`                 | `string`  | The unique system identifier of the audio device.        |
| <a id="property-device_setting_id"></a> `device_setting_id` | `string`  | The settings identifier for this device.                 |
| <a id="property-device_state"></a> `device_state`           | `string`  | The current state of the device (e.g. active, disabled). |
| <a id="property-display_name"></a> `display_name`           | `string`  | The human-readable display name of the audio device.     |
