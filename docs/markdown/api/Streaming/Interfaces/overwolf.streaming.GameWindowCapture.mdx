Overwolf APIs / overwolf / overwolf / streaming / GameWindowCapture

Game window capture options.

## Properties

| Property                                                                  | Type      | Description                                                                                               |
| ------------------------------------------------------------------------- | --------- | --------------------------------------------------------------------------------------------------------- |
| <a id="property-capture_overwolf_windows"></a> `capture_overwolf_windows` | `boolean` | Whether to capture Overwolf windows on top of the game. Default value is taken from the Overwolf Settings |
| <a id="property-enable_when_available"></a> `enable_when_available`       | `boolean` | Enable game window capture when the game window is available. Disabled by default.                        |
