Overwolf APIs / overwolf / overwolf / streaming / StopStreamingEvent

Event data fired when a stream stops.

## Properties

| Property                                              | Type      | Description                                            |
| ----------------------------------------------------- | --------- | ------------------------------------------------------ |
| <a id="property-duration"></a> `duration`             | `number`  | The total duration of the stream in milliseconds.      |
| <a id="property-extra"></a> `extra`                   | `string`  | Extra metadata associated with the stream session.     |
| <a id="property-file_path"></a> `file_path`           | `string`  | The path to the recorded video file, if applicable.    |
| <a id="property-last_file_path"></a> `last_file_path` | `string`  | The path to the last video file segment written.       |
| <a id="property-osbuild"></a> `osBuild`               | `string`  | The OS build string of the system.                     |
| <a id="property-osversion"></a> `osVersion`           | `string`  | The OS version string of the system.                   |
| <a id="property-split"></a> `split`                   | `boolean` | Whether the recording was split into multiple files.   |
| <a id="property-stream_id"></a> `stream_id`           | `number`  | The ID of the stream that stopped.                     |
| <a id="property-total_frames"></a> `total_frames`     | `number`  | The total number of frames captured during the stream. |
| <a id="property-url"></a> `url`                       | `string`  | The URL the stream was broadcasting to, if applicable. |
