Overwolf APIs / overwolf / overwolf / streaming / StreamingVideoEncoderx264Settings

Defines the configuration for an x264 encoder.

## Properties

| Property                                                    | Type                                                                                                  | Description                                                  |
| ----------------------------------------------------------- | ----------------------------------------------------------------------------------------------------- | ------------------------------------------------------------ |
| <a id="property-keyframe_interval"></a> `keyframe_interval` | `number`                                                                                              | Defines the number of frames after which to send a keyframe. |
| <a id="property-preset"></a> `preset?`                      | `StreamEncoderPreset_x264`           | Defines which preset the encoder should use.                 |
| <a id="property-rate_control"></a> `rate_control?`          | `StreamEncoderRateControl_x264` | Defines the rate control mode the encoder should use.        |
