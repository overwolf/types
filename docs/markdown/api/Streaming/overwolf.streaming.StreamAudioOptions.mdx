Overwolf APIs / overwolf.benchmarking / overwolf / streaming / StreamAudioOptions

Stream audio options.

## Properties

| Property                                                     | Type                                           | Description                                                                                                      |
| ------------------------------------------------------------ | ---------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| <a id="property-game"></a> ~~`game?`~~                       | `GameAudioDevice`       | Defines the game volume as applied to the stream.                                                                |
| <a id="property-game_volume"></a> ~~`game_volume?`~~         | `number`                                       | Defines the game volume as applied to the stream in a range of 0 to 100.                                         |
| <a id="property-mic"></a> ~~`mic?`~~                         | `StreamDeviceVolume` | Defines the microphone volume as applied to the stream.                                                          |
| <a id="property-mic_volume"></a> ~~`mic_volume?`~~           | `number`                                       | Defines the microphone volume as applied to the stream in a range of 0 to 100.                                   |
| <a id="property-separate_tracks"></a> ~~`separate_tracks?`~~ | `boolean`                                      | Enable multiple audio tracks: Track 1: Microphone + Desktop, Track 2: Desktop output, Track 3: Microphone input. |
