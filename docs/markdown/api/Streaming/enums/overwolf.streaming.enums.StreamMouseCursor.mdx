Overwolf APIs / overwolf / overwolf / streaming / enums / StreamMouseCursor

Controls when the mouse cursor is captured in a stream.

## Enumeration Members

| Enumeration Member                                        | Value           | Description                                             |
| --------------------------------------------------------- | --------------- | ------------------------------------------------------- |
| <a id="enumeration-member-both"></a> `both`               | `"both"`        | Capture the cursor in both game and desktop captures.   |
| <a id="enumeration-member-desktoponly"></a> `desktopOnly` | `"desktopOnly"` | Capture the cursor only when capturing the desktop.     |
| <a id="enumeration-member-gameonly"></a> `gameOnly`       | `"gameOnly"`    | Capture the cursor only when capturing the game window. |
| <a id="enumeration-member-none"></a> `none`               | `"none"`        | Never capture the mouse cursor.                         |
