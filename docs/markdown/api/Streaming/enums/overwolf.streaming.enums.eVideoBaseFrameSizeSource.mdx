Overwolf APIs / overwolf / overwolf / streaming / enums / eVideoBaseFrameSizeSource

Determines the source used to establish the base video frame size.

## Enumeration Members

| Enumeration Member                                | Value       | Description                                                          |
| ------------------------------------------------- | ----------- | -------------------------------------------------------------------- |
| <a id="enumeration-member-auto"></a> `Auto`       | `"Auto"`    | The base frame size is determined automatically.                     |
| <a id="enumeration-member-setting"></a> `Setting` | `"Setting"` | The base frame size is taken from the `StreamVideoOptions` settings. |
