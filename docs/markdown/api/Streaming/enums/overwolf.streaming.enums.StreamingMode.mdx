Overwolf APIs / overwolf / overwolf / streaming / enums / StreamingMode

Controls when an Overwolf window is included in a stream or recording.

## Enumeration Members

| Enumeration Member                                        | Value           | Description                                               |
| --------------------------------------------------------- | --------------- | --------------------------------------------------------- |
| <a id="enumeration-member-always"></a> `Always`           | `"Always"`      | Always include the window in the stream, even if hidden.  |
| <a id="enumeration-member-never"></a> `Never`             | `"Never"`       | Never include the window in the stream.                   |
| <a id="enumeration-member-whenvisible"></a> `WhenVisible` | `"WhenVisible"` | Include the window in the stream only when it is visible. |
