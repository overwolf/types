Defines a device volume and enablement settings.

## Extended by

- `GameAudioDevice`

## Properties

| Property                                         | Type      | Description                                         |
| ------------------------------------------------ | --------- | --------------------------------------------------- |
| <a id="property-device_id"></a> ~`device_id?`~ | `string`  | Defines the device ID to use.                       |
| <a id="property-enable"></a> ~`enable?`~       | `boolean` | Defines if the device is enabled.                   |
| <a id="property-volume"></a> ~`volume?`~       | `number`  | Defines the device volume in the range of 0 to 100. |
