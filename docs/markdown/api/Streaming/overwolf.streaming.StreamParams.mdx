Overwolf APIs / overwolf.benchmarking / overwolf / streaming / StreamParams

Represents the settings required to start a stream.

## Properties

| Property                                                 | Type                                                                                                            | Description                                               |
| -------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------- |
| <a id="property-audio"></a> ~`audio?`~                 | `StreamAudioOptions`                                                                  | Stream audio options.                                     |
| <a id="property-auth"></a> ~`auth?`~                   | `StreamAuthParams`                                                                      | Stream authorization data.                                |
| <a id="property-ingest_server"></a> ~`ingest_server?`~ | `StreamIngestServer`                                                                  | Information on the server that is being streamed to.      |
| <a id="property-max_quota_gb"></a> ~`max_quota_gb?`~   | `number`                                                                                                        | Max media folder size in GB. Deprecated                   |
| <a id="property-peripherals"></a> ~`peripherals?`~     | `StreamPeripheralsCaptureOptions`                                        | Defines how peripherals (i.e. mouse cursor) are streamed. |
| <a id="property-quota"></a> ~`quota?`~                 | `StreamQuotaParams`                                                                    | Quota information                                         |
| <a id="property-replay_type"></a> ~`replay_type?`~     | `Video` | The replay type to use.                                   |
| <a id="property-stream_info"></a> ~`stream_info?`~     | `StreamInfo`                                                                                  | The basic stream information.                             |
| <a id="property-video"></a> ~`video?`~                 | `StreamVideoOptions`                                                                  | Stream video options.                                     |
