Parameters required to initiate a video upload to a social platform.

## Properties

| Property                                  | Type     | Description                                             |
| ----------------------------------------- | -------- | ------------------------------------------------------- |
| <a id="property-filepath"></a> `filePath` | `string` | The local file system path to the video file to upload. |
| <a id="property-id"></a> `id`             | `string` | A unique identifier for this upload operation.          |
