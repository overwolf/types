Overwolf APIs / overwolf / overwolf / social / discord / ShareParameters

Parameters for sharing media to a Discord channel.

## Properties

| Property                                                                  | Type                                                                                                    | Description                                                                                                                                                                                                          |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <a id="property-channelid"></a> `channelId`                               | `string`                                                                                                | The ID of the Discord channel to share to.                                                                                                                                                                           |
| <a id="property-events"></a> `events?`                                    | `string`[]                                                                                              | An array of event labels associated with the captured clip.                                                                                                                                                          |
| <a id="property-file"></a> `file?`                                        | `string`                                                                                                | The file to share. Note: Since version 0.153, the "file" param is optional when calling overwolf.social.discord.share(). Instead, you can use the "message" param to include a URL of a file that you want to share. |
| <a id="property-gameclassid"></a> `gameClassId?`                          | `number`                                                                                                | The class ID of the associated game.                                                                                                                                                                                 |
| <a id="property-gametitle"></a> `gameTitle?`                              | `string`                                                                                                | The title of the associated game.                                                                                                                                                                                    |
| <a id="property-id"></a> `id?`                                            | `string`                                                                                                | An optional identifier for the share operation.                                                                                                                                                                      |
| <a id="property-message"></a> `message`                                   | `string`                                                                                                | The text message to accompany the shared media.                                                                                                                                                                      |
| <a id="property-metadata"></a> `metadata?`                                | `any`                                                                                                   | Extra information about the game session (How is this used?)                                                                                                                                                         |
| <a id="property-trimming"></a> `trimming?`                                | `VideoCompositionSegment` | An object containing start time and end time for the desired VideoCompositionSegment                                                                                                                                 |
| <a id="property-useoverwolfnotifications"></a> `useOverwolfNotifications` | `boolean`                                                                                               | Whether to display Overwolf notifications during the share.                                                                                                                                                          |
