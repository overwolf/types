Overwolf APIs / overwolf / overwolf / social / discord / Channel

Represents a Discord channel within a guild.

## Properties

| Property                                                            | Type                                                                    | Description                                                |
| ------------------------------------------------------------------- | ----------------------------------------------------------------------- | ---------------------------------------------------------- |
| <a id="property-guild_id"></a> `guild_id`                           | `string`                                                                | The ID of the guild this channel belongs to.               |
| <a id="property-id"></a> `id`                                       | `string`                                                                | The unique channel ID.                                     |
| <a id="property-name"></a> `name`                                   | `string`                                                                | The name of the channel.                                   |
| <a id="property-parent_id"></a> `parent_id?`                        | `string`                                                                | The ID of the parent category channel, if any.             |
| <a id="property-permission_overwrites"></a> `permission_overwrites` | `PermissionOverwrite`[]                      | The list of permission overwrites applied to this channel. |
| <a id="property-type"></a> `type`                                   | `number`                                                                | The channel type as a numeric value.                       |
| <a id="property-user_post_permission"></a> `user_post_permission`   | `PostPermission` | The current user's post permission level for this channel. |
