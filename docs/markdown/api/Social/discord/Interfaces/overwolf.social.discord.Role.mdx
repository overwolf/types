Represents a Discord role within a guild.

## Properties

| Property                                        | Type     | Description                             |
| ----------------------------------------------- | -------- | --------------------------------------- |
| <a id="property-id"></a> `id`                   | `string` | The unique role ID.                     |
| <a id="property-name"></a> `name`               | `string` | The name of the role.                   |
| <a id="property-permissions"></a> `permissions` | `number` | The permissions bitfield for this role. |
