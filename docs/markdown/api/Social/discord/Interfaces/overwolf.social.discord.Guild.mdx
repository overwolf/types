Overwolf APIs / overwolf / overwolf / social / discord / Guild

Represents a Discord guild (server).

## Properties

| Property                                   | Type                 | Description                                           |
| ------------------------------------------ | -------------------- | ----------------------------------------------------- |
| <a id="property-icon"></a> `icon?`         | `string`             | The guild's icon hash, if set.                        |
| <a id="property-id"></a> `id`              | `string`             | The unique guild ID.                                  |
| <a id="property-name"></a> `name`          | `string`             | The name of the guild.                                |
| <a id="property-owner_id"></a> `owner_id?` | `string`             | The ID of the guild owner, if available.              |
| <a id="property-roles"></a> `roles?`       | `Role`[] | The list of roles defined in the guild, if available. |
