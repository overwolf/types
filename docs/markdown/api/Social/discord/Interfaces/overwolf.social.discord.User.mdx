Overwolf APIs / overwolf / overwolf / social / discord / User

Represents a Discord user account.

## Properties

| Property                                            | Type      | Description                                                      |
| --------------------------------------------------- | --------- | ---------------------------------------------------------------- |
| <a id="property-avatar"></a> `avatar?`              | `string`  | The user's avatar hash, if set.                                  |
| <a id="property-discriminator"></a> `discriminator` | `number`  | The user's discriminator number (the four digits after the `#`). |
| <a id="property-email"></a> `email`                 | `string`  | The user's email address.                                        |
| <a id="property-id"></a> `id`                       | `string`  | The unique Discord user ID.                                      |
| <a id="property-username"></a> `username`           | `string`  | The user's Discord username.                                     |
| <a id="property-verified"></a> `verified`           | `boolean` | Whether the user's email address has been verified.              |
