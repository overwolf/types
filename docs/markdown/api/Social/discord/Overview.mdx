# Overview

| API | Description |
| --- | --- |
| [overwolf.social.discord.getChannels](./overwolf.social.discord.getChannels) | If the user is currently logged into Discord, this will return the channels of the given `guildId`, for which the user has privileges to share images/videos to. Otherwise, an error is returned |
| [overwolf.social.discord.getGuilds](./overwolf.social.discord.getGuilds) | If the user is currently logged into Discord, this will return the guilds that the user is registered to. Otherwise, an error is returned |
| [overwolf.social.discord.getUserInfo](./overwolf.social.discord.getUserInfo) | If the user is currently logged into Discord, this will return user information. Otherwise, an error is returned. |
| [overwolf.social.discord.performLogout](./overwolf.social.discord.performLogout) | Performs a "strong" sign out of Discord, so that even if the user performs a login via the Overwolf Settings / Accounts page, they will be considered signed out. |
| [overwolf.social.discord.performUserLogin](./overwolf.social.discord.performUserLogin) | Opens the login dialog. There is no callback for this method and the only way to know if the user signed in is via `onLoginStateChanged`. |
| [overwolf.social.discord.post](./overwolf.social.discord.post) | `void` |
| [overwolf.social.discord.share](./overwolf.social.discord.share) | If the user is currently logged into Discord, this will perform the media share (image or video).Possible errors that can occur:- Disconnected (user isn't signed in)- MissingFile (trying to share a missing file)- UnsupportedFile (trying to share an unsupported format)- ExceedsMaxSize (the file is too large: > 8 MB for images, > 100 MBfor videos) |
| [overwolf.social.discord.shareEx](./overwolf.social.discord.shareEx) | `void` |
| [overwolf.social.discord.Channel](./overwolf.social.discord.Channel) | - |
| [overwolf.social.discord.GetChannelsResult](./overwolf.social.discord.GetChannelsResult) | - |
| [overwolf.social.discord.GetGuildsResult](./overwolf.social.discord.GetGuildsResult) | - |
| [overwolf.social.discord.Guild](./overwolf.social.discord.Guild) | - |
| [overwolf.social.discord.PermissionOverwrite](./overwolf.social.discord.PermissionOverwrite) | - |
| [overwolf.social.discord.PostParameters](./overwolf.social.discord.PostParameters) | - |
| [overwolf.social.discord.Role](./overwolf.social.discord.Role) | - |
| [overwolf.social.discord.ShareParameters](./overwolf.social.discord.ShareParameters) | - |
| [overwolf.social.discord.SocialShareProgress](./overwolf.social.discord.SocialShareProgress) | - |
| [overwolf.social.discord.SocialShareResult](./overwolf.social.discord.SocialShareResult) | - |
| [overwolf.social.discord.User](./overwolf.social.discord.User) | - |
| [overwolf.social.discord.onLoginStateChanged](./overwolf.social.discord.onLoginStateChanged) | Fired when the user's login state changes. |
| [overwolf.social.discord.cancelUpload](./overwolf.social.discord.cancelUpload) | `void` |
| [overwolf.social.discord.getDisabledServices](./overwolf.social.discord.getDisabledServices) | Checks which of the supported sharing services are disabled or enabled. |
