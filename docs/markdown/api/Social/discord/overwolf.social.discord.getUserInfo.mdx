Overwolf APIs / overwolf.benchmarking / overwolf / social / discord / getUserInfo

```ts
function getUserInfo(callback: CallbackFunction<GetUserInfoResult<User>>): void;
```

If the user is currently logged into Discord, this will return user
information. Otherwise, an error is returned.

## Parameters

| Parameter  | Type                                                                                                                                                                                                | Description                                                       |
| ---------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------- |
| `callback` | `CallbackFunction`\<`GetUserInfoResult`\<`User`\>\> | Will contain user information or error if the request has failed. |

## Returns

`void`
