Overwolf APIs / overwolf / overwolf / social / discord / enums / PostPermission

Represents the posting permissions available for a Discord channel.

## Enumeration Members

| Enumeration Member                          | Value | Description                                  |
| ------------------------------------------- | ----- | -------------------------------------------- |
| <a id="enumeration-member-file"></a> `File` | `2`   | Permission to post files (images or videos). |
| <a id="enumeration-member-none"></a> `None` | `0`   | No posting permission.                       |
| <a id="enumeration-member-text"></a> `Text` | `1`   | Permission to post text messages.            |
