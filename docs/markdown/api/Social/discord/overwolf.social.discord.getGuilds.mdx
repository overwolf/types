Overwolf APIs / overwolf.benchmarking / overwolf / social / discord / getGuilds

```ts
function getGuilds(callback: CallbackFunction<GetGuildsResult>): void;
```

If the user is currently logged into Discord, this will return the guilds
that the user is registered to. Otherwise, an error is returned

## Parameters

| Parameter  | Type                                                                                                                                                  | Description                                                                 |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------- |
| `callback` | `CallbackFunction`\<`GetGuildsResult`\> | Will contain guild (server) information or error if the request has failed. |

## Returns

`void`
