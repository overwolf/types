Overwolf APIs / overwolf / overwolf / social / youtube / shareEx

```ts
function shareEx(
  discordShareParams: ShareParameters,
  resultCallback: CallbackFunction<SocialShareResult>,
  progressCallback: CallbackFunction<SocialShareProgress>,
): void;
```

Shares a video to YouTube with separate result and progress callbacks.

## Parameters

| Parameter            | Type                                                                                                                                                          | Description                                                      |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------- |
| `discordShareParams` | `ShareParameters`                                                                                                        | The share parameters.                                            |
| `resultCallback`     | `CallbackFunction`\<`SocialShareResult`\>     | Called with the final share result, including the published URL. |
| `progressCallback`   | `CallbackFunction`\<`SocialShareProgress`\> | Called periodically with upload progress updates.                |

## Returns

`void`
