Overwolf APIs / overwolf / overwolf / social / youtube / enums / YouTubePrivacy

The visibility setting for a video uploaded to YouTube.

## Enumeration Members

| Enumeration Member                                  | Value        | Description                                           |
| --------------------------------------------------- | ------------ | ----------------------------------------------------- |
| <a id="enumeration-member-private"></a> `Private`   | `"Private"`  | The video is visible only to the owner.               |
| <a id="enumeration-member-public"></a> `Public`     | `"Public"`   | The video is visible to everyone.                     |
| <a id="enumeration-member-unlisted"></a> `Unlisted` | `"Unlisted"` | The video is visible only to users who have the link. |
