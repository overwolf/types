# Overview

| API | Description |
| --- | --- |
| [overwolf.social.youtube.getUserInfo](./overwolf.social.youtube.getUserInfo) | If the user is currently logged into YouTube, this will return user information: &#123; avatar: "http://abs.twimg.com/sticky/...", id: "111111111112222222", name: "full name", screenName: "screenname123" &#125; Otherwise, an error is returned. |
| [overwolf.social.youtube.performLogout](./overwolf.social.youtube.performLogout) | Performs a "strong" sign out of YouTube, so that even if the user performs a login via the Overwolf Settings / Accounts page, he will be considered signed out. |
| [overwolf.social.youtube.performUserLogin](./overwolf.social.youtube.performUserLogin) | Opens the login dialog. There is no callback for this method and the only way to know if the user signed in is via `onLoginStateChanged`. |
| [overwolf.social.youtube.share](./overwolf.social.youtube.share) | If the user is currently logged into YouTube, this will perform the video share. |
| [overwolf.social.youtube.shareEx](./overwolf.social.youtube.shareEx) | `void` |
| [overwolf.social.youtube.ShareParameters](./overwolf.social.youtube.ShareParameters) | - |
| [overwolf.social.youtube.SocialShareProgress](./overwolf.social.youtube.SocialShareProgress) | - |
| [overwolf.social.youtube.SocialShareResult](./overwolf.social.youtube.SocialShareResult) | - |
| [overwolf.social.youtube.User](./overwolf.social.youtube.User) | - |
