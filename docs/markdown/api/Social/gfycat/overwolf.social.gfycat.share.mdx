Overwolf APIs / overwolf.benchmarking / overwolf / social / gfycat / share

```ts
function share(
  gfycatShareParams: ShareParameters,
  callback: CallbackFunction<Result>,
): void;
```

Possible errors that can occur:- Disconnected (user isn't signed in)-
MissingFile (trying to share a missing file)- UnsupportedFile (trying to
share an unsupported format)- ExceedsMaxSize (the file is too large: &gt; 8 MB
for images, &gt; 100 MBfor videos)

## Parameters

| Parameter           | Type                                                                                                                                            | Description                                     |
| ------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------- |
| `gfycatShareParams` | `ShareParameters`                                                                                          | The share parameters. See GfycatShareParameters |
| `callback`          | `CallbackFunction`\&lt;`Result`\&gt; | Will contain the status of the request.         |

## Returns

`void`
