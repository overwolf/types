Overwolf APIs / overwolf / overwolf / media / Webcam

Represents a connected webcam device.

## Properties

| Property                          | Type     | Description                                   |
| --------------------------------- | -------- | --------------------------------------------- |
| <a id="property-id"></a> `id`     | `string` | The unique identifier of the webcam device.   |
| <a id="property-name"></a> `name` | `string` | The human-readable name of the webcam device. |
| <a id="property-path"></a> `path` | `string` | The device path of the webcam.                |
