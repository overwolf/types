Overwolf APIs / overwolf / overwolf / media / CropParams

Parameters for cropping an image or capture frame.

## Properties

| Property                              | Type     | Description                                         |
| ------------------------------------- | -------- | --------------------------------------------------- |
| <a id="property-height"></a> `height` | `number` | The height of the crop region in pixels.            |
| <a id="property-width"></a> `width`   | `number` | The width of the crop region in pixels.             |
| <a id="property-x"></a> `x`           | `number` | The horizontal offset of the crop region in pixels. |
| <a id="property-y"></a> `y`           | `number` | The vertical offset of the crop region in pixels.   |
