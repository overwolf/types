A time range within a video, used to specify segments for composition.

## Properties

| Property                                    | Type     | Description                                    |
| ------------------------------------------- | -------- | ---------------------------------------------- |
| <a id="property-endtime"></a> `endTime`     | `number` | The end time of the segment in milliseconds.   |
| <a id="property-starttime"></a> `startTime` | `number` | The start time of the segment in milliseconds. |
