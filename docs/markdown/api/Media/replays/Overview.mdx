# Overview

Use this API to record, manage, and retrieve video replays and auto-highlights from game sessions. Replay capture runs in the background; call `turnOn` to start buffering and `capture` or `startCapture`/`stopCapture` to save clips.

## Functions

| API | Description |
| --- | --- |
| [overwolf.media.replays.addWatermark](./Functions/overwolf.media.replays.addWatermark) | Adds a video/image watermark to a video. |
| [overwolf.media.replays.capture](./Functions/overwolf.media.replays.capture) | Starts capturing a replay to a file. A replay id will be returned in the callback which is needed to finish capturing the replay. You can only call this method if replay mode is on and no other replay is currently being captured to a file. |
| [overwolf.media.replays.getHighlightsFeatures](./Functions/overwolf.media.replays.getHighlightsFeatures) | Get supported auto highlights features for a game |
| [overwolf.media.replays.getState](./Functions/overwolf.media.replays.getState) | Returns whether replay capturing is turned on or off. |
| [overwolf.media.replays.setReplaysSubFolder](./Functions/overwolf.media.replays.setReplaysSubFolder) | change target sub folder of current running replay provider |
| [overwolf.media.replays.startCapture](./Functions/overwolf.media.replays.startCapture) | Starts capturing a replay to a file. A replay id will be returned in the callback which is needed to finish capturing the replay. You can only call this method if replay mode is on and no other replay is currently being captured to a file. |
| [overwolf.media.replays.stopCapture](./Functions/overwolf.media.replays.stopCapture) | Finishes capturing a replay and returns a url to the created video file. You can only call this method if replay mode is on and using a valid id of a replay being captured to a file. |
| [overwolf.media.replays.turnOff](./Functions/overwolf.media.replays.turnOff) | Turns off background replay capturing. Call this as soon as you no longer interesting in capturing, in order to free up resources. |
| [overwolf.media.replays.turnOn](./Functions/overwolf.media.replays.turnOn) | Turns on background replay capturing. Without calling it first, you will not be able to create video replays. Notice that turning on replay capturing will consume system resources so use it wisely.buffer_length defines the amount of time in milliseconds to have captured in the memory at all times. |
