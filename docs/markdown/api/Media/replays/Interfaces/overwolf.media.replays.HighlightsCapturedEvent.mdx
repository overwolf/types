Overwolf APIs / overwolf / overwolf / media / replays / HighlightsCapturedEvent

Event data fired when an auto-highlight clip is captured.

## Properties

| Property                                                                | Type                             | Description                                                                     |
| ----------------------------------------------------------------------- | -------------------------------- | ------------------------------------------------------------------------------- |
| <a id="property-duration"></a> `duration`                               | `number`                         | Duration of the highlight clip in milliseconds.                                 |
| <a id="property-events"></a> `events`                                   | `string`[]                       | Array of event type strings that triggered this highlight.                      |
| <a id="property-game_id"></a> `game_id`                                 | `number`                         | The game ID for which the highlight was captured.                               |
| <a id="property-match_id"></a> `match_id`                               | `string`                         | Identifier of the match.                                                        |
| <a id="property-match_internal_id"></a> `match_internal_id`             | `string`                         | Internal identifier of the match.                                               |
| <a id="property-match_start_time"></a> `match_start_time`               | `number`                         | Unix timestamp (ms) when the match started.                                     |
| <a id="property-media_path"></a> `media_path`                           | `string`                         | File system path to the highlight video.                                        |
| <a id="property-media_path_encoded"></a> `media_path_encoded`           | `string`                         | URL-encoded file system path to the highlight video.                            |
| <a id="property-media_url"></a> `media_url`                             | `string`                         | Overwolf media URL to the highlight video.                                      |
| <a id="property-raw_events"></a> `raw_events`                           | `raw_events`[] | Array of raw event objects with type and timing.                                |
| <a id="property-replay_video_start_time"></a> `replay_video_start_time` | `number`                         | Unix timestamp (ms) of the start of the replay video containing this highlight. |
| <a id="property-session_id"></a> `session_id`                           | `string`                         | Identifier of the current session.                                              |
| <a id="property-session_start_time"></a> `session_start_time`           | `number`                         | Unix timestamp (ms) when the session started.                                   |
| <a id="property-start_time"></a> `start_time`                           | `number`                         | Unix timestamp (ms) when the highlight clip starts.                             |
| <a id="property-thumbnail_encoded_path"></a> `thumbnail_encoded_path`   | `string`                         | URL-encoded file system path to the highlight thumbnail.                        |
| <a id="property-thumbnail_path"></a> `thumbnail_path`                   | `string`                         | File system path to the highlight thumbnail.                                    |
| <a id="property-thumbnail_url"></a> `thumbnail_url`                     | `string`                         | Overwolf media URL to the highlight thumbnail.                                  |
