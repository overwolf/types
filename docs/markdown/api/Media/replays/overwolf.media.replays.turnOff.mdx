Overwolf APIs / overwolf.benchmarking / overwolf / media / replays / turnOff

```ts
function turnOff(callback: CallbackFunction<TurnOffResult>): void;
```

Turns off background replay capturing. Call this as soon as you no longer
interesting in capturing, in order to free up resources.

## Parameters

| Parameter  | Type                                                                                                                                              | Description                                                              |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------ |
| `callback` | `CallbackFunction`\<`TurnOffResult`\> | A callback function which will be called with the status of the request. |

## Returns

`void`
