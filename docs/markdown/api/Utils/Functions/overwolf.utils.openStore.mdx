```ts
function openStore(param: OpenStoreParams): void;
```

Opens the requested app's profile/login/subscription page in the Overwolf
Appstore.

## Parameters

| Parameter | Type                                                   | Description               |
| --------- | ------------------------------------------------------ | ------------------------- |
| `param`   | `OpenStoreParams` | The requested store page. |

## Returns

`void`
