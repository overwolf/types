Overwolf APIs / overwolf.benchmarking / overwolf / utils / openWindowsExplorer

```ts
function openWindowsExplorer(
  url: string,
  callback: CallbackFunction<Result>,
): void;
```

Opens Windows Explorer and selects a file received as an Overwolf media
url.

## Parameters

| Parameter  | Type                                                                                                                                | Description                                 |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------- |
| `url`      | `string`                                                                                                                            | An overwolf media url (overwolf: /  / media / \*) |
| `callback` | `CallbackFunction`\&lt;`Result`\&gt; | Called with the result of the request.      |

## Returns

`void`
