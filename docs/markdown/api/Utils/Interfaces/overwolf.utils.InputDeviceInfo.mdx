Information about a connected input device (keyboard, mouse, gamepad, etc.).

## Properties

| Property                              | Type     | Description                                  |
| ------------------------------------- | -------- | -------------------------------------------- |
| <a id="property-id"></a> `id`         | `number` | Numeric identifier for the input device.     |
| <a id="property-type"></a> `type`     | `string` | Type of input device (e.g. keyboard, mouse). |
| <a id="property-vendor"></a> `vendor` | `number` | Vendor ID of the input device.               |
