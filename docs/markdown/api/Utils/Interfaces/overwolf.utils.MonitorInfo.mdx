Overwolf APIs / overwolf / overwolf / utils / MonitorInfo

Information about a monitor as reported by the system.

## Properties

| Property                                      | Type      | Description                                   |
| --------------------------------------------- | --------- | --------------------------------------------- |
| <a id="property-dpix"></a> `Dpix`             | `number`  | Horizontal DPI of the monitor.                |
| <a id="property-dpiy"></a> `Dpiy`             | `number`  | Vertical DPI of the monitor.                  |
| <a id="property-ismain"></a> `IsMain`         | `boolean` | Whether this is the main/primary monitor.     |
| <a id="property-location"></a> `Location`     | `string`  | Physical or logical location identifier.      |
| <a id="property-name"></a> `Name`             | `string`  | Monitor name.                                 |
| <a id="property-resolution"></a> `Resolution` | `string`  | Monitor resolution string (e.g. "1920x1080"). |
