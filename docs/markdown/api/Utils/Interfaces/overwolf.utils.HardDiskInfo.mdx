Information about a hard disk or storage device in the system.

## Properties

| Property                                | Type      | Description                              |
| --------------------------------------- | --------- | ---------------------------------------- |
| <a id="property-caption"></a> `Caption` | `string`  | Display name/caption of the disk.        |
| <a id="property-isssd"></a> `IsSsd`     | `boolean` | Whether the drive is a solid-state disk. |
| <a id="property-size"></a> `Size`       | `number`  | Total disk size in bytes.                |
