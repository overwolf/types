```ts
function getMonitorsList(
  callback: CallbackFunction<getMonitorsListResult>,
): void;
```

bounds, and names.

| Parameter  | Type                                                                                                                                                        | Description                     |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------- |
| `callback` | `CallbackFunction`\&lt;`getMonitorsListResult`\&gt; | Called with the monitors array. |
