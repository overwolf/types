| Property                                  | Type                                                              | Description                           |

| <a id="property-croptype"></a> `cropType` | `AppLogoCrop` | How the logo image should be cropped. |
| <a id="property-url"></a> `url`           | `string`                                                          | URL of the replacement logo image.    |
