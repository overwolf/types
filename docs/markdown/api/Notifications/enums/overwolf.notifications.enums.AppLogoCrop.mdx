Overwolf APIs / overwolf / overwolf / notifications / enums / AppLogoCrop

Controls how the app logo is cropped inside the toast notification.

## Enumeration Members

| Enumeration Member                                | Value       | Description                            |
| ------------------------------------------------- | ----------- | -------------------------------------- |
| <a id="enumeration-member-circle"></a> `Circle`   | `"Circle"`  | Crop the logo into a circle.           |
| <a id="enumeration-member-default"></a> `Default` | `"Default"` | Use the platform default cropping.     |
| <a id="enumeration-member-none"></a> `None`       | `"None"`    | Display the logo without any cropping. |
