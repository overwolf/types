| <a id="enumeration-member-deleted"></a> `Deleted`       | `"Deleted"`    | The target resource has been removed.          |
| <a id="enumeration-member-registered"></a> `Registered` | `"Registered"` | The watcher has been successfully registered.  |
| <a id="enumeration-member-renamed"></a> `Renamed`       | `"Renamed"`    | The target resource has been moved or renamed. |
