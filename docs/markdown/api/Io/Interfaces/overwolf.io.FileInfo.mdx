## Properties

| Property                                      | Type      | Description                                     |
| --------------------------------------------- | --------- | ----------------------------------------------- |
| <a id="property-eof"></a> `eof`               | `boolean` | `true` if the end of the file has been reached. |
| <a id="property-position"></a> `position`     | `number`  | The current byte position within the file.      |
| <a id="property-totallines"></a> `totalLines` | `number`  | The total number of lines read so far.          |
| <a id="property-totalread"></a> `totalRead`   | `number`  | The total number of bytes read so far.          |
