Overwolf APIs / overwolf.io / ReadFileOptions

## Properties

| Property                                              | Type        | Description                                                                          |
| ----------------------------------------------------- | ----------- | ------------------------------------------------------------------------------------ |
| <a id="property-encoding"></a> `encoding`             | `eEncoding` | The character encoding to use when reading the file.                                 |
| <a id="property-maxbytestoread"></a> `maxBytesToRead` | `number`    | The maximum number of bytes to read from the file. Pass `0` to read the entire file. |
| <a id="property-offset"></a> `offset`                 | `number`    | The byte offset within the file at which to begin reading.                           |
