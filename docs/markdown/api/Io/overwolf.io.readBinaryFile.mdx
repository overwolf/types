Overwolf APIs / overwolf.benchmarking / overwolf / io / readBinaryFile

```ts
function readBinaryFile(
  path: string,
  options: ReadFileOptions,
  callback: CallbackFunction<ReadBinaryFileResult>,
): void;
```

Reads a file's contents and returns an array of byte values.
This function is extremely slow! Use it only for small files.

## Parameters

| Parameter  | Type                                                                                                                                                      | Description                                     |
| ---------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------- |
| `path`     | `string`                                                                                                                                                  | The target path.                                |
| `options`  | `ReadFileOptions`                                                                                                    | Describes the different options to read a file. |
| `callback` | `CallbackFunction`\&lt;`ReadBinaryFileResult`\&gt; | result callback.                                |

## Returns

`void`
