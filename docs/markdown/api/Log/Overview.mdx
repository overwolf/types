# Overview

| API | Description |
| --- | --- |
| [overwolf.log.error](./overwolf.log.error) | Writes error level log message to the common log. |
| [overwolf.log.info](./overwolf.log.info) | Writes info level log message to the common log. |
| [overwolf.log.onSupportedEncodersUpdated](./overwolf.log.onSupportedEncodersUpdated) | Fired upon support encoder list updated. |
| [overwolf.log.verbose](./overwolf.log.verbose) | Writes verbose (debug) level log message to the common log. |
| [overwolf.log.warning](./overwolf.log.warning) | Writes warning level log message to the common log. |
