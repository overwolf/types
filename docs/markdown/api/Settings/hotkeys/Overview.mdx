# Overview

| API | Description |
| --- | --- |
| [overwolf.settings.hotkeys.assign](./overwolf.settings.hotkeys.assign) | Assign global hotkey for the current extension, OR, if a gameId is specified, assign/unassign a dedicated hotkey. |
| [overwolf.settings.hotkeys.get](./overwolf.settings.hotkeys.get) | Returns the hotkey assigned for the current extension in all the games. |
| [overwolf.settings.hotkeys.getAllApps](./overwolf.settings.hotkeys.getAllApps) | Returns the hotkeys assigned for all installed extensions + the platform, in all the games. |
| [overwolf.settings.hotkeys.unassign](./overwolf.settings.hotkeys.unassign) | Unassign global hotkey for the current extension, OR, if a gameId is specified, assign/unassign a dedicated hotkey. |
| [overwolf.settings.hotkeys.update](./overwolf.settings.hotkeys.update) | `void` |
| [overwolf.settings.hotkeys.AssignHotkeyObject](./overwolf.settings.hotkeys.AssignHotkeyObject) | - |
| [overwolf.settings.hotkeys.GetAllAssignedHotkeysResult](./overwolf.settings.hotkeys.GetAllAssignedHotkeysResult) | - |
| [overwolf.settings.hotkeys.GetAssignedHotkeyResult](./overwolf.settings.hotkeys.GetAssignedHotkeyResult) | - |
| [overwolf.settings.hotkeys.HotkeyModifiers](./overwolf.settings.hotkeys.HotkeyModifiers) | - |
| [overwolf.settings.hotkeys.IHotkey](./overwolf.settings.hotkeys.IHotkey) | - |
| [overwolf.settings.hotkeys.OnChangedEvent](./overwolf.settings.hotkeys.OnChangedEvent) | - |
| [overwolf.settings.hotkeys.OnHoldEvent](./overwolf.settings.hotkeys.OnHoldEvent) | - |
| [overwolf.settings.hotkeys.OnPressedEvent](./overwolf.settings.hotkeys.OnPressedEvent) | - |
| [overwolf.settings.hotkeys.UnassignHotkeyObject](./overwolf.settings.hotkeys.UnassignHotkeyObject) | - |
| [overwolf.settings.hotkeys.UpdateHotkeyObject](./overwolf.settings.hotkeys.UpdateHotkeyObject) | - |
| [overwolf.settings.hotkeys.onHold](./overwolf.settings.hotkeys.onHold) | Fired only for hotkeys that are set in the manifest as hold. |
| [overwolf.settings.hotkeys.onPressed](./overwolf.settings.hotkeys.onPressed) | Fired for hotkeys that are NOT set as hold hotkeys. |
| [overwolf.settings.hotkeys.onAutoLaunchEnablementChanged](./overwolf.settings.hotkeys.onAutoLaunchEnablementChanged) | Fired when auto launch is enabled or disabled for a game. |
