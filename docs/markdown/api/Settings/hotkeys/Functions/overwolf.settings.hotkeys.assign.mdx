Overwolf APIs / overwolf / overwolf / settings / hotkeys / assign

```ts
function assign(
  hotkey: AssignHotkeyObject,
  callback: CallbackFunction<Result>,
): void;
```

Assign global hotkey for the current extension, OR, if a gameId is specified, assign/unassign a dedicated hotkey.

## Parameters

| Parameter  | Type                                                                                                                                            | Description                                                                     |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------- |
| `hotkey`   | `AssignHotkeyObject`                                                                                    | The hotkey to assign, including the name, key binding, and optional game scope. |
| `callback` | `CallbackFunction`\<`Result`\> | Called with the result of the assignment request.                               |

## Returns

`void`
