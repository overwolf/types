Overwolf APIs / overwolf / overwolf / settings / hotkeys / HotkeyModifiers

Specifies which modifier keys are active in a hotkey combination.

## Properties

| Property                             | Type      | Description                           |
| ------------------------------------ | --------- | ------------------------------------- |
| <a id="property-alt"></a> `alt?`     | `boolean` | Whether the Alt modifier is active.   |
| <a id="property-ctrl"></a> `ctrl?`   | `boolean` | Whether the Ctrl modifier is active.  |
| <a id="property-shift"></a> `shift?` | `boolean` | Whether the Shift modifier is active. |
