Overwolf APIs / overwolf.benchmarking / overwolf / settings / games / getAutoLaunchEnabled

```ts
function getAutoLaunchEnabled(
  gameClassId: number,
  callback: CallbackFunction<AutolaunchEnabledResult>,
): void;
```

Returns the current Auto-Launch enabled setting for the calling app in a given game (gameClassId).

## Parameters

| Parameter     | Type                                                                                                                                                                  | Description                                     |
| ------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------- |
| `gameClassId` | `number`                                                                                                                                                              | the game id for which the flag is retrieved for |
| `callback`    | `CallbackFunction`\<`AutolaunchEnabledResult`\> | -                                               |

## Returns

`void`
