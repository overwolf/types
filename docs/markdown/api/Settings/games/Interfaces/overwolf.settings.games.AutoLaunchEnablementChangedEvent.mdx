Event data describing a change in auto-launch enablement for a game.

## Properties

| Property                                | Type      | Description                                           |
| --------------------------------------- | --------- | ----------------------------------------------------- |
| <a id="property-appid"></a> `appId`     | `string`  | The ID of the app whose auto-launch setting changed.  |
| <a id="property-enabled"></a> `enabled` | `boolean` | The new auto-launch enabled state.                    |
| <a id="property-gameid"></a> `gameId`   | `number`  | The ID of the game whose auto-launch setting changed. |
