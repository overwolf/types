# Overview

| API | Description |
| --- | --- |
| [overwolf.settings.games.getAutoLaunchEnabled](./overwolf.settings.games.getAutoLaunchEnabled) | Returns the current Auto-Launch enabled setting for the calling app in a given game (gameClassId). |
| [overwolf.settings.games.getOverlayEnabled](./overwolf.settings.games.getOverlayEnabled) | Returns the current Overlay setting for the given game (if any exist). |
| [overwolf.settings.games.setAutoLaunchEnabled](./overwolf.settings.games.setAutoLaunchEnabled) | Sets the current Auto-Launch enabled setting for the calling app in a given game (gameClassId). |
| [overwolf.settings.games.AutolaunchEnabledResult](./overwolf.settings.games.AutolaunchEnabledResult) | - |
| [overwolf.settings.games.AutoLaunchEnablementChangedEvent](./overwolf.settings.games.AutoLaunchEnablementChangedEvent) | - |
| [overwolf.settings.games.GameClassResult](./overwolf.settings.games.GameClassResult) | - |
| [overwolf.settings.games.OverlayEnabledResult](./overwolf.settings.games.OverlayEnabledResult) | - |
| [overwolf.settings.games.OverlayEnablementChangedEvent](./overwolf.settings.games.OverlayEnablementChangedEvent) | - |
| [overwolf.settings.games.onOverlayEnablementChanged](./overwolf.settings.games.onOverlayEnablementChanged) | Fired when the overlay is enabled or disabled for a game. |
| [overwolf.settings.games.OnHotKeyChanged](./overwolf.settings.games.OnHotKeyChanged) | Fired when a hotkey is modified. Apps will only be notified ofhotkey changes that relate to them. |
