Overwolf APIs / overwolf / overwolf / settings / setFpsSettings

```ts
function setFpsSettings(
  settings: FpsSettings,
  callback: CallbackFunction<Result>,
): void;
```

Sets the state (on/off), position, offset (in pixels) and scale [0, 1] of
the Fps control.

| ---------- | ----------------------------------------------------------------------------------------------------------------------------------- | ----------- |
| `settings` | `FpsSettings`                                                                                      | -           |
| `callback` | `CallbackFunction`\<`Result`\> | -           |
