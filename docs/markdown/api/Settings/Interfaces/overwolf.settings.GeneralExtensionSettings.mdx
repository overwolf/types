Overwolf APIs / overwolf / overwolf / settings / GeneralExtensionSettings

General settings for an Overwolf extension.

## Properties

| Property                                                                     | Type      | Description                                                             |
| ---------------------------------------------------------------------------- | --------- | ----------------------------------------------------------------------- |
| <a id="property-auto_launch_with_overwolf"></a> `auto_launch_with_overwolf?` | `boolean` | Whether the extension should launch automatically when Overwolf starts. |
| <a id="property-channel"></a> `channel?`                                     | `string`  | The update channel the extension subscribes to.                         |
| <a id="property-exit_overwolf_on_exit"></a> `exit_overwolf_on_exit?`         | `boolean` | Whether Overwolf should exit when the extension exits.                  |
