Overwolf APIs / overwolf / overwolf / settings / HotKeyChangedEvent

Event data fired when a hotkey binding changes.

## Properties

| Property                                        | Type     | Description                                           |
| ----------------------------------------------- | -------- | ----------------------------------------------------- |
| <a id="property-description"></a> `description` | `string` | A human-readable description of the hotkey action.    |
| <a id="property-hotkey"></a> `hotkey`           | `string` | The new hotkey string.                                |
| <a id="property-source"></a> `source`           | `string` | The source extension or feature that owns the hotkey. |
