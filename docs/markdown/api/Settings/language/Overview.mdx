# Overview

The `overwolf.settings.language` namespace lets you retrieve the Overwolf client's current UI language and listen for language change events.

## Functions

| API | Description |
| --- | --- |
| [overwolf.settings.language.get](./Functions/overwolf.settings.language.get) | Returns the current language Overwolf is set to in a two-letter ISO 639-1 format. |

## Interfaces

| API | Description |
| --- | --- |
| [overwolf.settings.language.GetLanguageResult](./Interfaces/overwolf.settings.language.GetLanguageResult) | Result of a `get` call containing the current language code. |
| [overwolf.settings.language.LanguageChangedEvent](./Interfaces/overwolf.settings.language.LanguageChangedEvent) | Event data fired when the Overwolf UI language changes. |

## Variables

| API | Description |
| --- | --- |
| [overwolf.settings.language.onChanged](./Variables/overwolf.settings.language.onChanged) | Fired on hotkey setting change. |
