Overwolf APIs / overwolf / overwolf / settings / language / get

```ts
function get(callback: CallbackFunction<GetLanguageResult>): void;
```

Returns the current language Overwolf is set to in a two-letter ISO 639-1 format.

## Parameters

| Parameter  | Type                                                                                                                                                      | Description                            |
| ---------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------- |
| `callback` | `CallbackFunction`\<`GetLanguageResult`\> | Called with the current language code. |

## Returns

`void`
