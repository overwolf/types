# Overview

| API | Description |
| --- | --- |
| [paths.commonAppData](./Variables/paths.commonAppData) | The path to the common application data directory (e.g. `C:\ProgramData`). |
| [paths.commonDocuments](./Variables/paths.commonDocuments) | The path to the shared (all-users) Documents folder. |
| [paths.commonFiles](./Variables/paths.commonFiles) | The path to the common program files directory. |
| [paths.commonFilesX86](./Variables/paths.commonFilesX86) | The path to the 32-bit common program files directory. |
| [paths.desktop](./Variables/paths.desktop) | The path to the current user's Desktop folder. |
| [paths.documents](./Variables/paths.documents) | The path to the current user's Documents folder. |
| [paths.favorites](./Variables/paths.favorites) | The path to the current user's Favorites folder. |
| [paths.fonts](./Variables/paths.fonts) | The path to the Windows Fonts directory. |
| [paths.localAppData](./Variables/paths.localAppData) | The path to the current user's local application data directory (e.g. `%LOCALAPPDATA%`). |
| [paths.music](./Variables/paths.music) | The path to the current user's Music folder. |
| [paths.obsBin](./Variables/paths.obsBin) | The path to the OBS binary directory used by Overwolf. |
| [paths.overwolfInstallation](./Variables/paths.overwolfInstallation) | The path to the Overwolf installation directory. |
| [paths.overwolfInstallationWithVersion](./Variables/paths.overwolfInstallationWithVersion) | The path to the versioned Overwolf installation directory (includes the current version number). |
| [paths.pictures](./Variables/paths.pictures) | The path to the current user's Pictures folder. |
| [paths.programFiles](./Variables/paths.programFiles) | The path to the `Program Files` directory (e.g. `C:\Program Files`). |
| [paths.programFilesX86](./Variables/paths.programFilesX86) | The path to the 32-bit `Program Files` directory (e.g. `C:\Program Files (x86)`). |
| [paths.startMenu](./Variables/paths.startMenu) | The path to the current user's Start Menu folder. |
| [paths.system](./Variables/paths.system) | The path to the System32 directory. |
| [paths.systemX86](./Variables/paths.systemX86) | The path to the 32-bit SysWOW64 (or System32) directory. |
| [paths.videos](./Variables/paths.videos) | The path to the current user's Videos folder. |
| [paths.windows](./Variables/paths.windows) | The path to the Windows installation directory (e.g. `C:\Windows`). |
