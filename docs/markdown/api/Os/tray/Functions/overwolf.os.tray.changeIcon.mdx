## Call Signature

```ts
function changeIcon(path: string, callback: CallbackFunction<Result>): void;
```

Changes the tray icon to the image at the given file path.

### Parameters

| Parameter  | Type                                                                                                                                            | Description                          |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------ |
| `path`     | `string`                                                                                                                                        | The path to the new icon image file. |
| `callback` | `CallbackFunction`\<`Result`\> | Called with the result.              |

### Returns

`void`

## Call Signature

```ts
function changeIcon(
  space: StorageSpace,
  path: string,
  callback: CallbackFunction<Result>,
): void;
```

Changes the tray icon to the image at the given path within the specified storage space.

### Parameters

| Parameter  | Type                                                                                                                                            | Description                                                   |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------- |
| `space`    | `StorageSpace`                                           | The storage space where the icon file is located.             |
| `path`     | `string`                                                                                                                                        | The path to the new icon image file within the storage space. |
| `callback` | `CallbackFunction`\<`Result`\> | Called with the result.                                       |

### Returns

`void`
