Overwolf APIs / overwolf.os.tray / RegionInfo

Regional formatting preferences for the current user's locale.

## Properties

| Property                                                 | Type      | Description                                                              |
| -------------------------------------------------------- | --------- | ------------------------------------------------------------------------ |
| <a id="property-currency_symbol"></a> `currency_symbol?` | `string`  | The currency symbol used in the user's locale (e.g., `"$"`).             |
| <a id="property-date_format"></a> `date_format?`         | `string`  | The date format string used in the user's locale (e.g., `"MM/DD/YYYY"`). |
| <a id="property-is_metric"></a> `is_metric?`             | `boolean` | Whether the user's locale uses the metric system.                        |
| <a id="property-name"></a> `name?`                       | `string`  | The name of the user's locale or region.                                 |
| <a id="property-time_format"></a> `time_format?`         | `string`  | The time format string used in the user's locale (e.g., `"HH:mm"`).      |
