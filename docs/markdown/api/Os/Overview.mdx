# Overview

The `overwolf.os` namespace provides utilities for interacting with the host operating system, including regional settings and system tray integration (via `overwolf.os.tray`).

## Namespaces

| API | Description |
| --- | --- |
| [tray](./tray/Overview) | Use this API to create and manage a system tray icon for your Overwolf app, including context menus, icon changes, and click events. |

## Functions

| API | Description |
| --- | --- |
| [overwolf.os.critical](./Functions/overwolf.os.critical) | Writes error level log message to the common log. |
