Overwolf APIs / overwolf / overwolf / web / createWebSocket

```ts
function createWebSocket(
  connectionParams: WebSocketConnectionParams,
  callback: CallbackFunction<CreateWebSocketResult>,
): void;
```

Creates a WebSocket client connection to localhost/127.0.0.1, bypassing
certificate verification. Call `connect` on the returned `WebSocket`
instance to open the connection.

## Parameters

| Parameter          | Type                                                                                                                                                        | Description                                                                       |
| ------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| `connectionParams` | `WebSocketConnectionParams`                                                                                  | Connection parameters including port, protocol list, and optional credentials.    |
| `callback`         | `CallbackFunction`\<`CreateWebSocketResult`\> | Called with the result, including the new `WebSocket` client instance on success. |

## Returns

`void`
