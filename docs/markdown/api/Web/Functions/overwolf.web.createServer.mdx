Overwolf APIs / overwolf / overwolf / web / createServer

```ts
function createServer(
  port: number,
  callback: CallbackFunction<CreateServerResult>,
): void;
```

Creates a local HTTP web server on the specified port. The returned
`WebServer` instance must have its `listen` method called before it will
accept connections.

## Parameters

| Parameter  | Type                                                                                                                                                  | Description                                                                |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| `port`     | `number`                                                                                                                                              | The port number to bind the server to.                                     |
| `callback` | `CallbackFunction`\<`CreateServerResult`\> | Called with the result, including the new `WebServer` instance on success. |

## Returns

`void`
