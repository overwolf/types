Overwolf APIs / overwolf / overwolf / web / enums / HttpRequestMethods

The HTTP method to use when sending a fetch request.

## Enumeration Members

| Enumeration Member                              | Value      | Description                               |
| ----------------------------------------------- | ---------- | ----------------------------------------- |
| <a id="enumeration-member-delete"></a> `DELETE` | `"DELETE"` | Remove a resource.                        |
| <a id="enumeration-member-get"></a> `GET`       | `"GET"`    | Retrieve a resource.                      |
| <a id="enumeration-member-head"></a> `HEAD`     | `"HEAD"`   | Retrieve only the headers for a resource. |
| <a id="enumeration-member-patch"></a> `PATCH`   | `"PATCH"`  | Apply a partial update to a resource.     |
| <a id="enumeration-member-post"></a> `POST`     | `"POST"`   | Submit data to a resource.                |
| <a id="enumeration-member-put"></a> `PUT`       | `"PUT"`    | Replace a resource entirely.              |
