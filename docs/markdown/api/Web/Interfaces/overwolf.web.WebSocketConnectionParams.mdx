Overwolf APIs / overwolf / overwolf / web / WebSocketConnectionParams

Connection parameters used to create a `WebSocket` client.

## Properties

| Property                                        | Type       | Description                                           |
| ----------------------------------------------- | ---------- | ----------------------------------------------------- |
| <a id="property-credentials"></a> `credentials` | `object`   | Optional credentials for basic authentication.        |
| `credentials.password`                          | `string`   | -                                                     |
| `credentials.username`                          | `string`   | -                                                     |
| <a id="property-port"></a> `port`               | `number`   | The port number to connect to.                        |
| <a id="property-protocols"></a> `protocols`     | `string`[] | A list of sub-protocols to negotiate with the server. |
| <a id="property-secured"></a> `secured`         | `boolean`  | Whether to use a secure (`wss://`) connection.        |
