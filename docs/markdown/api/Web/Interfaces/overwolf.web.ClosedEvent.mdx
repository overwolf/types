Describes the reason a WebSocket connection was closed.

## Properties

| Property                              | Type     | Description                                                     |
| ------------------------------------- | -------- | --------------------------------------------------------------- |
| <a id="property-code"></a> `code`     | `number` | The WebSocket close status code.                                |
| <a id="property-reason"></a> `reason` | `string` | A human-readable explanation for why the connection was closed. |
