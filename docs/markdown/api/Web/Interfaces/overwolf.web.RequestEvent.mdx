Overwolf APIs / overwolf / overwolf / web / RequestEvent

Describes an incoming HTTP request received by a `WebServer`.

## Properties

| Property                                        | Type     | Description                        |
| ----------------------------------------------- | -------- | ---------------------------------- |
| <a id="property-content"></a> `content`         | `string` | The request body content.          |
| <a id="property-contenttype"></a> `contentType` | `string` | The MIME type of the request body. |
| <a id="property-url"></a> `url`                 | `string` | The request URL path.              |
