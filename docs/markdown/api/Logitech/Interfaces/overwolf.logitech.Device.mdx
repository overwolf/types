Overwolf APIs / overwolf / overwolf / logitech / Device

Information about a connected Logitech peripheral device.

## Properties

| Property                                          | Type     | Description                                     |
| ------------------------------------------------- | -------- | ----------------------------------------------- |
| <a id="property-lightingid"></a> `lightingId`     | `number` | The numeric ID of the device's lighting zone.   |
| <a id="property-lightingname"></a> `lightingName` | `string` | The display name of the device's lighting zone. |
| <a id="property-name"></a> `name`                 | `string` | The display name of the device.                 |
| <a id="property-pid"></a> `pid`                   | `number` | The USB product ID of the device.               |
| <a id="property-typeid"></a> `typeId`             | `number` | The numeric type identifier of the device.      |
| <a id="property-typename"></a> `typeName`         | `string` | The display name of the device type.            |
