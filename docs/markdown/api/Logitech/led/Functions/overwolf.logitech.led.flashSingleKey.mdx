Overwolf APIs / overwolf / overwolf / logitech / led / flashSingleKey

```ts
function flashSingleKey(
  keyName: KeyboardNames,
  redPercentage: number,
  greenPercentage: number,
  bluePercentage: number,
  milliSecondsDuration: number,
  milliSecondsInterval: number,
  callback: CallbackFunction<Result>,
): void;
```

Flashes a single key.

## Parameters

| Parameter              | Type                                                                                                                                            | Description                                |
| ---------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------ |
| `keyName`              | `KeyboardNames`                                                                           | The key name. For a list of key names see  |
| `redPercentage`        | `number`                                                                                                                                        | Red percentage (0 - 100)                   |
| `greenPercentage`      | `number`                                                                                                                                        | Green percentage (0 - 100)                 |
| `bluePercentage`       | `number`                                                                                                                                        | Blue percentage (0 - 100)                  |
| `milliSecondsDuration` | `number`                                                                                                                                        | The duration to flash in milliseconds.     |
| `milliSecondsInterval` | `number`                                                                                                                                        | The interval for flashes in milliseconds.  |
| `callback`             | `CallbackFunction`\<`Result`\> | A callback with the result of the request. |

## Returns

`void`
