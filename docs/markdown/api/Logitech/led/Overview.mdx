# Overview

| API | Description |
| --- | --- |
| [overwolf.logitech.led.flashLighting](./overwolf.logitech.led.flashLighting) | Flashes the lighting on the device. |
| [overwolf.logitech.led.flashSingleKey](./overwolf.logitech.led.flashSingleKey) | Flashes a single key. |
| [overwolf.logitech.led.init](./overwolf.logitech.led.init) | Initializes the LED API. |
| [overwolf.logitech.led.pulseLighting](./overwolf.logitech.led.pulseLighting) | Pulses the lighting on the device. |
| [overwolf.logitech.led.pulseSingleKey](./overwolf.logitech.led.pulseSingleKey) | Pulses a single key. |
| [overwolf.logitech.led.restoreLighting](./overwolf.logitech.led.restoreLighting) | Restores the lightning to the last previously saved state. |
| [overwolf.logitech.led.restoreLightingForKey](./overwolf.logitech.led.restoreLightingForKey) | Restores a previously saved lighting for a specific key. |
| [overwolf.logitech.led.saveCurrentLighting](./overwolf.logitech.led.saveCurrentLighting) | Saves the current lighting. |
| [overwolf.logitech.led.saveLightingForKey](./overwolf.logitech.led.saveLightingForKey) | Saves the current lighting of a specific key. |
| [overwolf.logitech.led.setLighting](./overwolf.logitech.led.setLighting) | Sets the lighting for the entire device. |
| [overwolf.logitech.led.setLightingForKeyWithHidCode](./overwolf.logitech.led.setLightingForKeyWithHidCode) | Sets the lighting for a specific key by HID code. |
| [overwolf.logitech.led.setLightingForKeyWithKeyName](./overwolf.logitech.led.setLightingForKeyWithKeyName) | Sets the lighting for a specific key by key name. |
| [overwolf.logitech.led.setLightingForKeyWithQuartzCode](./overwolf.logitech.led.setLightingForKeyWithQuartzCode) | Sets the lighting for a specific key by quartz code. |
| [overwolf.logitech.led.setLightingForKeyWithScanCode](./overwolf.logitech.led.setLightingForKeyWithScanCode) | Sets the lighting for a specific key by scan code. |
| [overwolf.logitech.led.setLightingFromBitmap](./overwolf.logitech.led.setLightingFromBitmap) | Sets the lighting from an overwolf-extension:// or overwolf-media:// url. The file must be 21x6. |
| [overwolf.logitech.led.setTargetDevice](./overwolf.logitech.led.setTargetDevice) | Sets the target devices to use. |
| [overwolf.logitech.led.shutdown](./overwolf.logitech.led.shutdown) | Shuts down the API. |
| [overwolf.logitech.led.stopEffects](./overwolf.logitech.led.stopEffects) | Stops ongoing pulse/flash effects. |
| [overwolf.logitech.led.stopEffectsOnKey](./overwolf.logitech.led.stopEffectsOnKey) | Stops ongoing pulse/flash effects on a specific key. |
| [overwolf.logitech.led.ByteArray](./overwolf.logitech.led.ByteArray) | - |
| [overwolf.logitech.led.getDevices](./overwolf.logitech.led.getDevices) | Gets the currently installed Logitech devices. |
