# Overview

Use this API to control Logitech LED peripheral lighting from your Overwolf app. Supports per-key and zone-based RGB effects on compatible Logitech keyboards, mice, and other devices.

## Functions

| API | Description |
| --- | --- |
| [overwolf.logitech.led.flashLighting](./Functions/overwolf.logitech.led.flashLighting) | Flashes the lighting on the device. |
| [overwolf.logitech.led.flashSingleKey](./Functions/overwolf.logitech.led.flashSingleKey) | Flashes a single key. |
| [overwolf.logitech.led.getDevices](./Functions/overwolf.logitech.led.getDevices) | Gets the currently installed Logitech devices. |
| [overwolf.logitech.led.init](./Functions/overwolf.logitech.led.init) | Initializes the LED API. |
| [overwolf.logitech.led.pulseLighting](./Functions/overwolf.logitech.led.pulseLighting) | Pulses the lighting on the device. |
| [overwolf.logitech.led.pulseSingleKey](./Functions/overwolf.logitech.led.pulseSingleKey) | Pulses a single key. |
| [overwolf.logitech.led.restoreLighting](./Functions/overwolf.logitech.led.restoreLighting) | Restores the lightning to the last previously saved state. |
| [overwolf.logitech.led.restoreLightingForKey](./Functions/overwolf.logitech.led.restoreLightingForKey) | Restores a previously saved lighting for a specific key. |
| [overwolf.logitech.led.saveCurrentLighting](./Functions/overwolf.logitech.led.saveCurrentLighting) | Saves the current lighting. |
| [overwolf.logitech.led.saveLightingForKey](./Functions/overwolf.logitech.led.saveLightingForKey) | Saves the current lighting of a specific key. |
| [overwolf.logitech.led.setLighting](./Functions/overwolf.logitech.led.setLighting) | Sets the lighting for the entire device. |
| [overwolf.logitech.led.setLightingForKeyWithHidCode](./Functions/overwolf.logitech.led.setLightingForKeyWithHidCode) | Sets the lighting for a specific key by HID code. |
| [overwolf.logitech.led.setLightingForKeyWithKeyName](./Functions/overwolf.logitech.led.setLightingForKeyWithKeyName) | Sets the lighting for a specific key by key name. |
| [overwolf.logitech.led.setLightingForKeyWithQuartzCode](./Functions/overwolf.logitech.led.setLightingForKeyWithQuartzCode) | Sets the lighting for a specific key by quartz code. |
| [overwolf.logitech.led.setLightingForKeyWithScanCode](./Functions/overwolf.logitech.led.setLightingForKeyWithScanCode) | Sets the lighting for a specific key by scan code. |
| [overwolf.logitech.led.setLightingFromBitmap](./Functions/overwolf.logitech.led.setLightingFromBitmap) | Sets the lighting from an overwolf-extension:// or overwolf-media:// url. The file must be 21x6. |
| [overwolf.logitech.led.setTargetDevice](./Functions/overwolf.logitech.led.setTargetDevice) | Sets the target devices to use. |
| [overwolf.logitech.led.shutdown](./Functions/overwolf.logitech.led.shutdown) | Shuts down the API. |
| [overwolf.logitech.led.stopEffects](./Functions/overwolf.logitech.led.stopEffects) | Stops ongoing pulse/flash effects. |
| [overwolf.logitech.led.stopEffectsOnKey](./Functions/overwolf.logitech.led.stopEffectsOnKey) | Stops ongoing pulse/flash effects on a specific key. |
