# Overview

| API | Description |
| --- | --- |
| [CrossAppCampaign](./overwolf.campaigns.crossapp.CrossAppCampaign) | Container that represent a shared data parameters. |
| [CrossAppCampaignConversion](./overwolf.campaigns.crossapp.CrossAppCampaignConversion) | - |
| [getAvailableActions](./overwolf.campaigns.crossapp.getAvailableActions) | Receive all cross-app actions that target the currently running extension. |
| [GetCrossAppAvailableActionsResult](./overwolf.campaigns.crossapp.GetCrossAppAvailableActionsResult) | See `overwolf.campaigns.crossapp.getAvailableActions` |
| [GetCrossAppConversionsResult](./overwolf.campaigns.crossapp.GetCrossAppConversionsResult) | See `overwolf.campaigns.crossapp.consumeConversions` |
| [reportConversion](./overwolf.campaigns.crossapp.reportConversion) | Submit new conversion for a cross-app campaign. |
| [set](./overwolf.campaigns.crossapp.set) | Initiate or modify a cross-app campaign action for this extension. You may modify an existing action by using the same id parameter - see CrossAppCampaign.id |
