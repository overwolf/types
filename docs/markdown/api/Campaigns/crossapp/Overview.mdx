# Overview

| API | Description |
| --- | --- |
| [overwolf.campaigns.crossapp.getAvailableActions](./overwolf.campaigns.crossapp.getAvailableActions) | Receive all cross-app actions that target the currently running extension. |
| [overwolf.campaigns.crossapp.reportConversion](./overwolf.campaigns.crossapp.reportConversion) | Submit new conversion for a cross-app campaign. |
| [overwolf.campaigns.crossapp.set](./overwolf.campaigns.crossapp.set) | Initiate or modify a cross-app campaign action for this extension. You may modify an existing action by using the same id parameter - see CrossAppCampaign.id |
| [overwolf.campaigns.crossapp.CrossAppCampaign](./overwolf.campaigns.crossapp.CrossAppCampaign) | Container that represent a shared data parameters. |
| [overwolf.campaigns.crossapp.CrossAppCampaignConversion](./overwolf.campaigns.crossapp.CrossAppCampaignConversion) | - |

| [overwolf.campaigns.crossapp.GetCrossAppConversionsResult](./overwolf.campaigns.crossapp.GetCrossAppConversionsResult) | See \|overwolf.campaigns.crossapp.consumeConversions\| |
