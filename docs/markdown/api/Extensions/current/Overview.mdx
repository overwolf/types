# Overview

| API | Description |
| --- | --- |
| [overwolf.extensions.current.generateUserEmailHashes](./Functions/overwolf.extensions.current.generateUserEmailHashes) | Generates SHA1, SHA256, and MD5 hashes from the given email address. |
| [overwolf.extensions.current.getExtraObject](./Functions/overwolf.extensions.current.getExtraObject) | Retrieves an extra object (providing external APIs) registered in the extension's manifest. |
| [overwolf.extensions.current.getManifest](./Functions/overwolf.extensions.current.getManifest) | Returns the current extension's manifest object. |
| [overwolf.extensions.current.getPhasedPercent](./Functions/overwolf.extensions.current.getPhasedPercent) | Returns the phased rollout percentage for the current or specified extension version. |
| [overwolf.extensions.current.setUserEmailHashes](./Functions/overwolf.extensions.current.setUserEmailHashes) | Sets pre-computed email hashes on the current user's profile. |
| [overwolf.extensions.current.GetExtraObjectResult](./Interfaces/overwolf.extensions.current.GetExtraObjectResult) | Result of `getExtraObject`, containing the requested extra object instance. |
| [overwolf.extensions.current.UserEmailHashes](./Interfaces/overwolf.extensions.current.UserEmailHashes) | A set of cryptographic hashes derived from a user's email address. |
