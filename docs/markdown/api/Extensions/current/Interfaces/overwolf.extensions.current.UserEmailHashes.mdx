| Property                              | Type     | Description                    |
| ------------------------------------- | -------- | ------------------------------ |
| <a id="property-md5"></a> `MD5`       | `string` | The MD5 hash of the email.     |
| <a id="property-sha1"></a> `SHA1`     | `string` | The SHA-1 hash of the email.   |
| <a id="property-sha256"></a> `SHA256` | `string` | The SHA-256 hash of the email. |
