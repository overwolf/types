## Properties

| Property                                                              | Type                                             | Description                                                                                                              |
| --------------------------------------------------------------------- | ------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------ |
| <a id="property-data"></a> `data`                                     | `WebAppSettings`           | A list of additional meta-data on the app                                                                                |
| <a id="property-manifest_version"></a> `manifest_version`             | `number`                                         | Targets the manifest version you are working on. Currently there is only one version, therefore this value is always "1" |
| <a id="property-max_rotation_log_files"></a> `max_rotation_log_files` | `number`                                         | Increase the app's log file rotation (defaults to 10, max is 40).                                                        |
| <a id="property-meta"></a> `meta`                                     | `Metadata`                       | Includes app metadata                                                                                                    |
| <a id="property-permissions"></a> `permissions`                       | `Permission`[] | An array of permissions that the app requires.                                                                           |
| <a id="property-type"></a> `type`                                     | `"WebApp"`                                       | Declares the type of application. Can only be "WebApp"                                                                   |
| <a id="property-uid"></a> `UID`                                       | `string`                                         | -                                                                                                                        |
