Event data fired when the extension has been updated to a new version.

## Properties

| Property                                | Type                                                               | Description                                       |
| --------------------------------------- | ------------------------------------------------------------------ | ------------------------------------------------- |
| <a id="property-state"></a> `state`     | `ExtensionUpdateState` | The current update state of the extension.        |
| <a id="property-version"></a> `version` | `string`                                                           | The version string of the newly installed update. |
