Overwolf APIs / overwolf / overwolf / extensions / sharedData / set

```ts
function set(
  appId: string,
  value: any,
  callback: CallbackFunction<Result>,
): void;
```

Used by the owner app to set data for the consumer app, by appId.

## Parameters

| Parameter  | Type                                                                                                                                            | Description                                              |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------- |
| `appId`    | `string`                                                                                                                                        | The app ID of the consumer extension to share data with. |
| `value`    | `any`                                                                                                                                           | The data to share with the consumer app.                 |
| `callback` | `CallbackFunction`\<`Result`\> | Called with the result of the operation.                 |

## Returns

`void`
