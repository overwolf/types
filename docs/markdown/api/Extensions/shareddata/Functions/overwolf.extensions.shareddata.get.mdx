Overwolf APIs / overwolf / overwolf / extensions / sharedData / get

```ts
function get(
  param: SharedDataParams,
  callback: CallbackFunction<GetResult>,
): void;
```

Used by the consumer app to get data set by the owner app.

## Parameters

| Parameter  | Type                                                                                                                                      | Description                                                    |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------- |
| `param`    | `SharedDataParams`                                                                                  | Filter parameters specifying the origin and/or target app IDs. |
| `callback` | `CallbackFunction`\<`GetResult`\> | Called with the result containing the shared data.             |

## Returns

`void`
