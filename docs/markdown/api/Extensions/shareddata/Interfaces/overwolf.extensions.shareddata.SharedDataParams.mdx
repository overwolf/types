Container that represent a shared data parameters.

## Properties

| Property                               | Type     | Description                                                                     |
| -------------------------------------- | -------- | ------------------------------------------------------------------------------- |
| <a id="property-origin"></a> `origin?` | `string` | The app ID of the extension that set the shared data (the owner).               |
| <a id="property-target"></a> `target?` | `string` | The app ID of the extension that should receive the shared data (the consumer). |
