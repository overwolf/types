Event data for when shared data is changed by the owner app.

## Properties

| Property                              | Type     | Description                                           |
| ------------------------------------- | -------- | ----------------------------------------------------- |
| <a id="property-data"></a> `data`     | `string` | The updated shared data value.                        |
| <a id="property-origin"></a> `origin` | `string` | The app ID of the extension that set the shared data. |
| <a id="property-target"></a> `target` | `string` | The app ID of the extension the data was shared with. |
