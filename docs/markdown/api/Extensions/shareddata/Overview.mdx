# Overview

Repairs the schema registration, for an extension where the manifest contains url_protocol

## Functions

| API | Description |
| --- | --- |
| [overwolf.extensions.shareddata.get](./Functions/overwolf.extensions.shareddata.get) | Used by the consumer app to get data set by the owner app. |
| [overwolf.extensions.shareddata.repairUrlProtocol](./Functions/overwolf.extensions.shareddata.repairUrlProtocol) | Repairs the schema registration, for an extension where the manifest contains url_protocol |
| [overwolf.extensions.shareddata.set](./Functions/overwolf.extensions.shareddata.set) | Used by the owner app to set data for the consumer app, by appId. |

## Interfaces

| API | Description |
| --- | --- |
| [overwolf.extensions.shareddata.GetResult](./Interfaces/overwolf.extensions.shareddata.GetResult) | Result of `get`. |
| [overwolf.extensions.shareddata.onChangedEvent](./Interfaces/overwolf.extensions.shareddata.onChangedEvent) | Event data for when shared data is changed by the owner app. |
| [overwolf.extensions.shareddata.SharedDataParams](./Interfaces/overwolf.extensions.shareddata.SharedDataParams) | Container that represent a shared data parameters. |
