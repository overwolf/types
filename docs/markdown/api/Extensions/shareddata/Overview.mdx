# Overview

| API | Description |
| --- | --- |
| [overwolf.extensions.shareddata.get](./overwolf.extensions.shareddata.get) | Used by the consumer app to get data set by the owner app. |
| [overwolf.extensions.shareddata.set](./overwolf.extensions.shareddata.set) | Used by the owner app to set data for the consumer app, by appId. |
| [overwolf.extensions.shareddata.GetResult](./overwolf.extensions.shareddata.GetResult) | - |
| [overwolf.extensions.shareddata.onChangedEvent](./overwolf.extensions.shareddata.onChangedEvent) | - |
| [overwolf.extensions.shareddata.SharedDataParams](./overwolf.extensions.shareddata.SharedDataParams) | Container that represent a shared data parameters. |
| [overwolf.extensions.shareddata.repairUrlProtocol](./overwolf.extensions.shareddata.repairUrlProtocol) | Repairs the schema registration, for an extension where the manifest contains url_protocol |
