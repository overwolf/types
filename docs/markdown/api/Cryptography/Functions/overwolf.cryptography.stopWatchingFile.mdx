Overwolf APIs / overwolf.cryptography / stopWatchingFile

```ts
function stopWatchingFile(
  path: string,
  callback: CallbackFunction<Result>,
): void;
```

Stops watching a file that was previously registered with `watchFile`.

## Parameters

| Parameter  | Type                                                                                  | Description                                     |
| ---------- | ------------------------------------------------------------------------------------- | ----------------------------------------------- |
| `path`     | `string`                                                                              | The absolute path of the file to stop watching. |
| `callback` | `CallbackFunction`\<`Result`\> | Called with the result of the operation.        |

## Returns

`void`
