# Overview

The `overwolf.gep` namespace exposes internal Game Events Provider metadata, primarily used for diagnostics and version reporting.

| API | Description |
| --- | --- |
| [overwolf.gep.GepInternal](./Type-aliases/overwolf.gep.GepInternal) | Internal GEP metadata, used for diagnostics. |
| [overwolf.gep.onShareFailed](./Variables/overwolf.gep.onShareFailed) | Fired when an error is returned from Reddit. |
