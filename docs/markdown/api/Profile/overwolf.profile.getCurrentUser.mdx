Overwolf APIs / overwolf.benchmarking / overwolf / profile / getCurrentUser

```ts
function getCurrentUser(callback: CallbackFunction<GetCurrentUserResult>): void;
```

Calls the given callback with the currently logged-in Overwolf user.

## Parameters

| Parameter  | Type                                                                                                                                                      | Description                                           |
| ---------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------- |
| `callback` | `CallbackFunction`\&lt;`GetCurrentUserResult`\&gt; | A function called with the current user, or an error. |

## Returns

`void`
