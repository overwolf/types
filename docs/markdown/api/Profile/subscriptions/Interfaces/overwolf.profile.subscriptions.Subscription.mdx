Overwolf APIs / overwolf / overwolf / profile / subscriptions / Subscription

Represents a single user subscription record.

## Properties

| Property                                  | Type                                                                          | Description                                                     |
| ----------------------------------------- | ----------------------------------------------------------------------------- | --------------------------------------------------------------- |
| <a id="property-exp"></a> `exp`           | `number`                                                                      | The Unix timestamp (in seconds) when this subscription expires. |
| <a id="property-expired"></a> `expired`   | `boolean`                                                                     | Whether the subscription has already expired.                   |
| <a id="property-extid"></a> `extid`       | `string`                                                                      | The extension ID this subscription belongs to.                  |
| <a id="property-grc"></a> `grc`           | `number`                                                                      | Grace period count for this subscription.                       |
| <a id="property-id"></a> `id`             | `number`                                                                      | The unique subscription record ID.                              |
| <a id="property-muid"></a> `muid`         | `string`                                                                      | The machine user ID.                                            |
| <a id="property-pid"></a> `pid`           | `number`                                                                      | The plan ID associated with this subscription.                  |
| <a id="property-planinfo"></a> `planInfo` | `Info`                                                            | Detailed information about the subscribed plan.                 |
| <a id="property-state"></a> `state`       | `SubscriptionState` | The current state of the subscription.                          |
| <a id="property-uid"></a> `uid`           | `string`                                                                      | The user ID of the subscriber.                                  |
