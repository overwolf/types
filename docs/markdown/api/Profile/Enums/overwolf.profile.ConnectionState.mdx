Overwolf APIs / overwolf / overwolf / profile / ConnectionState

Represents the current connection state of the Overwolf client to its services.

## Enumeration Members

| Enumeration Member                                            | Value             | Description                                    |
| ------------------------------------------------------------- | ----------------- | ---------------------------------------------- |
| <a id="enumeration-member-connecting"></a> `Connecting`       | `"Connecting"`    | The client is in the process of connecting.    |
| <a id="enumeration-member-disconnecting"></a> `Disconnecting` | `"Disconnecting"` | The client is in the process of disconnecting. |
| <a id="enumeration-member-offline"></a> `Offline`             | `"Offline"`       | The client is not connected.                   |
| <a id="enumeration-member-online"></a> `Online`               | `"Online"`        | The client is connected.                       |
| <a id="enumeration-member-unknown"></a> `Unknown`             | `"Unknown"`       | The connection state is not known.             |
