Overwolf APIs / overwolf / overwolf / profile / generateUserSessionToken

```ts
function generateUserSessionToken(
  callback: CallbackFunction<GenerateUserSessionTokenResult>,
): void;
```

Generates a short-lived session token for the currently logged-in user.

## Parameters

| Parameter  | Type                                                                                                                                                                          | Description                                              |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------- |
| `callback` | `CallbackFunction`\<`GenerateUserSessionTokenResult`\> | A function called with the generated token, or an error. |

## Returns

`void`
