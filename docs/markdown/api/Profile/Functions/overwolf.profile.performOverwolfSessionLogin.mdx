Overwolf APIs / overwolf / overwolf / profile / performOverwolfSessionLogin

```ts
function performOverwolfSessionLogin(
  token: string,
  callback: CallbackFunction<Result>,
): void;
```

Performs an Overwolf session login using a previously obtained token.

## Parameters

| Parameter  | Type                                                                                                                                | Description                                             |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------- |
| `token`    | `string`                                                                                                                            | The session token to authenticate with.                 |
| `callback` | `CallbackFunction`\<`Result`\> | A function called with the result of the login attempt. |

## Returns

`void`
