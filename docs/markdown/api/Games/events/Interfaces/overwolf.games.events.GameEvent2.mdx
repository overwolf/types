Overwolf APIs / overwolf / overwolf / games / events / GameEvent2

A typed game event whose name is constrained to keys of the dictionary type `T`.

## Type Parameters

| Type Parameter                                                                   | Default type |
| -------------------------------------------------------------------------------- | ------------ |
| `T` _extends_ `GameEventDictionary2` | `any`        |

## Properties

| Property                          | Type      | Description                                             |
| --------------------------------- | --------- | ------------------------------------------------------- |
| <a id="property-data"></a> `data` | `any`     | The event data payload.                                 |
| <a id="property-name"></a> `name` | keyof `T` | The name of the event, restricted to known keys of `T`. |
