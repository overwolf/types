Overwolf APIs / overwolf / overwolf / games / events / InfoUpdates2Event

Payload of the `onInfoUpdates2` event, pairing an info update with its feature name.

## Type Parameters

| Type Parameter                                    | Default type                     |
| ------------------------------------------------- | -------------------------------- |
| `Feature`                                         | `string`                         |
| `Info` _extends_ `InfoUpdate2` | `InfoUpdate2` |

## Properties

| Property                                | Type      | Description                                    |
| --------------------------------------- | --------- | ---------------------------------------------- |
| <a id="property-feature"></a> `feature` | `Feature` | The feature that produced this info update.    |
| <a id="property-info"></a> `info`       | `Info`    | The info update data for the relevant feature. |
