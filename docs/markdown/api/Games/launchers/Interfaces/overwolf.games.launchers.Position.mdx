Screen position and dimensions of a window.

## Properties

| Property                              | Type     | Description                                              |
| ------------------------------------- | -------- | -------------------------------------------------------- |
| <a id="property-height"></a> `height` | `number` | The height of the window in pixels.                      |
| <a id="property-left"></a> `left`     | `number` | The distance from the left edge of the screen in pixels. |
| <a id="property-top"></a> `top`       | `number` | The distance from the top edge of the screen in pixels.  |
| <a id="property-width"></a> `width`   | `number` | The width of the window in pixels.                       |
