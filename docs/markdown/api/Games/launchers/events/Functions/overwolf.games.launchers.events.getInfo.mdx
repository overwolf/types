```ts
function getInfo(
  launcherClassId: number,
  callback: CallbackFunction<GetInfoResult<any>>,
): void;
```

Gets the current game info.

## Parameters

| Parameter         | Type                                                                                                                                                             | Description                            |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------- |
| `launcherClassId` | `number`                                                                                                                                                         | The class ID of the target launcher.   |
| `callback`        | `CallbackFunction`\<`GetInfoResult`\<`any`\>\> | Called with the current launcher info. |

## Returns

`void`
