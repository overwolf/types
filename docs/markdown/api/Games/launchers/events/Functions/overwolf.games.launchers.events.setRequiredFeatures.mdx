Overwolf APIs / overwolf / overwolf / games / launchers / events / setRequiredFeatures

```ts
function setRequiredFeatures(
  launcherClassId: number,
  features: string[],
  callback: CallbackFunction<SetRequiredFeaturesResult>,
): void;
```

Sets the required features from the provider.

## Parameters

| Parameter         | Type                                                                                                                                                                            | Description                            |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------- |
| `launcherClassId` | `number`                                                                                                                                                                        | The class ID of the target launcher.   |
| `features`        | `string`[]                                                                                                                                                                      | A string array of features to utilize. |
| `callback`        | `CallbackFunction`\<`SetRequiredFeaturesResult`\> | Called with success or failure state.  |

## Returns

`void`
