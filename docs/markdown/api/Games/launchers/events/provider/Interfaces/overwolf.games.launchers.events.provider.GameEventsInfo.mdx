Overwolf APIs / overwolf / overwolf / games / launchers / events / provider / GameEventsInfo

A single game event info update entry containing feature, category, key, and value.

## Properties

| Property                                  | Type     | Description                                         |
| ----------------------------------------- | -------- | --------------------------------------------------- |
| <a id="property-category"></a> `category` | `string` | The category of the info update within the feature. |
| <a id="property-feature"></a> `feature`   | `string` | The feature that produced this info update.         |
| <a id="property-key"></a> `key`           | `string` | The key identifying the specific info field.        |
| <a id="property-value"></a> `value`       | `string` | The new value for the info field.                   |
