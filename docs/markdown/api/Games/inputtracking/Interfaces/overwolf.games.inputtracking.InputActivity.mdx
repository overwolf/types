Overwolf APIs / overwolf / overwolf / games / inputTracking / InputActivity

Aggregated input activity statistics for the current game session.

## Properties

| Property                                  | Type     | Description                                                                                        |
| ----------------------------------------- | -------- | -------------------------------------------------------------------------------------------------- |
| <a id="property-apm"></a> `apm`           | `number` | Actions per minute across all input devices.                                                       |
| <a id="property-atime"></a> `aTime`       | `number` | Total active (non-idle) time in milliseconds.                                                      |
| <a id="property-itime"></a> `iTime`       | `number` | Total idle time in milliseconds.                                                                   |
| <a id="property-keyboard"></a> `keyboard` | `object` | Keyboard activity stats including total key presses and a per-key breakdown.                       |
| `keyboard.keys`                           | `any`    | -                                                                                                  |
| `keyboard.total`                          | `number` | -                                                                                                  |
| <a id="property-mouse"></a> `mouse`       | `object` | Mouse activity stats including total clicks, cursor distance traveled, and a per-button breakdown. |
| `mouse.dist`                              | `number` | -                                                                                                  |
| `mouse.keys`                              | `any`    | -                                                                                                  |
| `mouse.total`                             | `number` | -                                                                                                  |
