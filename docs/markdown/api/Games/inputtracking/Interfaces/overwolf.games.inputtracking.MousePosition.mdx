Overwolf APIs / overwolf / overwolf / games / inputTracking / MousePosition

The current position of the mouse cursor.

## Properties

| Property                              | Type      | Description                                                                               |
| ------------------------------------- | --------- | ----------------------------------------------------------------------------------------- |
| <a id="property-handle"></a> `handle` | `object`  | Native OS window handle for the window under the cursor.                                  |
| `handle.value`                        | `number`  | -                                                                                         |
| <a id="property-ongame"></a> `onGame` | `boolean` | Whether the cursor is positioned over the game window (as opposed to an Overwolf widget). |
| <a id="property-x"></a> `x`           | `number`  | Horizontal cursor position in pixels.                                                     |
| <a id="property-y"></a> `y`           | `number`  | Vertical cursor position in pixels.                                                       |
