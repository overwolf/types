## Properties

| Property                              | Type      | Description                                                     |
| ------------------------------------- | --------- | --------------------------------------------------------------- |
| <a id="property-key"></a> `key`       | `string`  | The virtual key code of the key that was pressed or released.   |
| <a id="property-ongame"></a> `onGame` | `boolean` | Whether the key event occurred while the game window had focus. |
