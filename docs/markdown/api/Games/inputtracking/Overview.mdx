# Overview

| API | Description |
| --- | --- |
| [overwolf.games.inputtracking.getActivityInformation](./overwolf.games.inputtracking.getActivityInformation) | Returns the input activity information. The information includes presses for keyboard/mouse, total session time, idle time and actions-per-minute. This information resets between game executions. |
| [overwolf.games.inputtracking.getEyeTrackingInformation](./overwolf.games.inputtracking.getEyeTrackingInformation) | Returns the eye tracking information. The information includes gaze points, fixations, user presence (screen/keyboard/other) and minimap glances. This information resets between game executions. |
| [overwolf.games.inputtracking.getMatchActivityInformation](./overwolf.games.inputtracking.getMatchActivityInformation) | Returns the input activity information (similar to `getActivityInformation`). However, when this is supported, it will return data only for the latestmatch of the current game |
| [overwolf.games.inputtracking.getMousePosition](./overwolf.games.inputtracking.getMousePosition) | Returns the input last mouse position in game. the data includes the mouse position and a boolean stating whether the keypress was on a game or on an Overwolf widget (onGame). |
| [overwolf.games.inputtracking.init](./overwolf.games.inputtracking.init) | `void` |
| [overwolf.games.inputtracking.pauseEyeTracking](./overwolf.games.inputtracking.pauseEyeTracking) | Eye tracking data trakcing will pause, and stop collect Eye tracking data until resumeEyeTracking will be called. |
| [overwolf.games.inputtracking.resumeEyeTracking](./overwolf.games.inputtracking.resumeEyeTracking) | Resume collecting Eye tracking data. |
| [overwolf.games.inputtracking.GetActivityResult](./overwolf.games.inputtracking.GetActivityResult) | - |
| [overwolf.games.inputtracking.GetMousePositionResult](./overwolf.games.inputtracking.GetMousePositionResult) | - |
| [overwolf.games.inputtracking.InputActivity](./overwolf.games.inputtracking.InputActivity) | - |
| [overwolf.games.inputtracking.KeyEvent](./overwolf.games.inputtracking.KeyEvent) | - |
| [overwolf.games.inputtracking.MouseEvent](./overwolf.games.inputtracking.MouseEvent) | - |
| [overwolf.games.inputtracking.MousePosition](./overwolf.games.inputtracking.MousePosition) | - |
| [overwolf.games.inputtracking.WheelEvent](./overwolf.games.inputtracking.WheelEvent) | - |
| [overwolf.games.inputtracking.onKeyDown](./overwolf.games.inputtracking.onKeyDown) | Fired when a keyboard key has been pressed. Event information is similar to `onKeyUp`. |
| [overwolf.games.inputtracking.onKeyUp](./overwolf.games.inputtracking.onKeyUp) | Fired when a keyboard key has been released. The event information includes the virtual key code (key) and a boolean stating whether the keypress was on a game or on an Overwolf widget (onGame). |
| [overwolf.games.inputtracking.onMouseDown](./overwolf.games.inputtracking.onMouseDown) | Fired a mouse key has been pressed. Event information is similar to `onMouseUp`. |
| [overwolf.games.inputtracking.onMouseUp](./overwolf.games.inputtracking.onMouseUp) | Fired when a mouse key has been released. The event information includes whether the left or white mouse button was clicked(button), x and y coordinates (x, y) and a boolean stating whether the keypress was on a game or on an Overwolf widget (onGame). |
