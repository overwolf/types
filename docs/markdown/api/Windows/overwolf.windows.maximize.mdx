Overwolf APIs / overwolf.benchmarking / overwolf / windows / maximize

```ts
function maximize(
  windowId: string,
  callback?: CallbackFunction<WindowIdResult>,
): void;
```

Maximizes the window.

## Parameters

| Parameter   | Type                                                                                                                                          | Description                               |
| ----------- | --------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------- |
| `windowId`  | `string`                                                                                                                                      | The id or name of the window to maximize. |
| `callback?` | `CallbackFunction`\&lt;`WindowIdResult`\&gt; | Called after the window is maximized.     |

## Returns

`void`
