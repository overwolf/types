# Overview

| API | Description |
| --- | --- |
| [overwolf.windows.mediaplayerelement.create](./overwolf.windows.mediaplayerelement.create) | Creates a media player a places it in the given location with given dimensions. |
| [overwolf.windows.mediaplayerelement.getProgress](./overwolf.windows.mediaplayerelement.getProgress) | Gets the current progress, in seconds, of the playback. |
| [overwolf.windows.mediaplayerelement.pause](./overwolf.windows.mediaplayerelement.pause) | Pauses the current video. |
| [overwolf.windows.mediaplayerelement.play](./overwolf.windows.mediaplayerelement.play) | Plays the current video. |
| [overwolf.windows.mediaplayerelement.removeAllPlayers](./overwolf.windows.mediaplayerelement.removeAllPlayers) | Remove all media players created for this window. |
| [overwolf.windows.mediaplayerelement.resume](./overwolf.windows.mediaplayerelement.resume) | Resumes the current video. |
| [overwolf.windows.mediaplayerelement.seek](./overwolf.windows.mediaplayerelement.seek) | Seeks the current video to the given number of seconds. |
| [overwolf.windows.mediaplayerelement.setBounds](./overwolf.windows.mediaplayerelement.setBounds) | Relocates the media player to a given location with given dimensions. |
| [overwolf.windows.mediaplayerelement.setPlaybackSpeed](./overwolf.windows.mediaplayerelement.setPlaybackSpeed) | Sets the speed ratio of the playback. |
| [overwolf.windows.mediaplayerelement.setStretchMode](./overwolf.windows.mediaplayerelement.setStretchMode) | Sets the stretch mode of the player. |
| [overwolf.windows.mediaplayerelement.setVideo](./overwolf.windows.mediaplayerelement.setVideo) | Sets the current video to be played. |
| [overwolf.windows.mediaplayerelement.setVolume](./overwolf.windows.mediaplayerelement.setVolume) | Sets the volume. |
| [overwolf.windows.mediaplayerelement.stop](./overwolf.windows.mediaplayerelement.stop) | Stops the current video. |
| [overwolf.windows.mediaplayerelement.toBack](./overwolf.windows.mediaplayerelement.toBack) | Sends the media player to the back of the window. |
| [overwolf.windows.mediaplayerelement.toFront](./overwolf.windows.mediaplayerelement.toFront) | Sends the media player to the front of the window. |
| [overwolf.windows.mediaplayerelement.CreateResult](./overwolf.windows.mediaplayerelement.CreateResult) | - |
| [overwolf.windows.mediaplayerelement.GetProgressResult](./overwolf.windows.mediaplayerelement.GetProgressResult) | - |
| [overwolf.windows.mediaplayerelement.PlaybackEvent](./overwolf.windows.mediaplayerelement.PlaybackEvent) | - |
| [overwolf.windows.mediaplayerelement.SetVideoResult](./overwolf.windows.mediaplayerelement.SetVideoResult) | - |
| [overwolf.windows.mediaplayerelement.onPlaybackEnded](./overwolf.windows.mediaplayerelement.onPlaybackEnded) | Fired when playback ends. |
| [overwolf.windows.mediaplayerelement.onPlaybackError](./overwolf.windows.mediaplayerelement.onPlaybackError) | Fired when there was an error while trying to open a video. |
| [overwolf.windows.mediaplayerelement.onPlaybackPaused](./overwolf.windows.mediaplayerelement.onPlaybackPaused) | Fired when playback is paused. |
| [overwolf.windows.mediaplayerelement.onPlaybackStarted](./overwolf.windows.mediaplayerelement.onPlaybackStarted) | Fired when playback is starting/resuming. |
| [overwolf.windows.mediaplayerelement.onPlaybackStopped](./overwolf.windows.mediaplayerelement.onPlaybackStopped) | Fired when playback is stopped. |
| [overwolf.windows.mediaplayerelement.onScreenPropertyChanged](./overwolf.windows.mediaplayerelement.onScreenPropertyChanged) | Fired when native window (or OSR on desktop) moved to other monitoror when current monitor resolution changed |
