# Overview

Use this API to embed and control native media player elements within Overwolf windows for video playback.

## Functions

| API | Description |
| --- | --- |
| [overwolf.windows.mediaplayerelement.create](./Functions/overwolf.windows.mediaplayerelement.create) | Creates a media player a places it in the given location with given dimensions. |
| [overwolf.windows.mediaplayerelement.getProgress](./Functions/overwolf.windows.mediaplayerelement.getProgress) | Gets the current progress, in seconds, of the playback. |
| [overwolf.windows.mediaplayerelement.pause](./Functions/overwolf.windows.mediaplayerelement.pause) | Pauses the current video. |
| [overwolf.windows.mediaplayerelement.play](./Functions/overwolf.windows.mediaplayerelement.play) | Plays the current video. |
| [overwolf.windows.mediaplayerelement.removeAllPlayers](./Functions/overwolf.windows.mediaplayerelement.removeAllPlayers) | Remove all media players created for this window. |
| [overwolf.windows.mediaplayerelement.resume](./Functions/overwolf.windows.mediaplayerelement.resume) | Resumes the current video. |
| [overwolf.windows.mediaplayerelement.seek](./Functions/overwolf.windows.mediaplayerelement.seek) | Seeks the current video to the given number of seconds. |
| [overwolf.windows.mediaplayerelement.setBounds](./Functions/overwolf.windows.mediaplayerelement.setBounds) | Relocates the media player to a given location with given dimensions. |
| [overwolf.windows.mediaplayerelement.setPlaybackSpeed](./Functions/overwolf.windows.mediaplayerelement.setPlaybackSpeed) | Sets the speed ratio of the playback. |
| [overwolf.windows.mediaplayerelement.setStretchMode](./Functions/overwolf.windows.mediaplayerelement.setStretchMode) | Sets the stretch mode of the player. |
| [overwolf.windows.mediaplayerelement.setVideo](./Functions/overwolf.windows.mediaplayerelement.setVideo) | Sets the current video to be played. |
| [overwolf.windows.mediaplayerelement.setVolume](./Functions/overwolf.windows.mediaplayerelement.setVolume) | Sets the volume. |
| [overwolf.windows.mediaplayerelement.stop](./Functions/overwolf.windows.mediaplayerelement.stop) | Stops the current video. |
| [overwolf.windows.mediaplayerelement.toBack](./Functions/overwolf.windows.mediaplayerelement.toBack) | Sends the media player to the back of the window. |
| [overwolf.windows.mediaplayerelement.toFront](./Functions/overwolf.windows.mediaplayerelement.toFront) | Sends the media player to the front of the window. |

## Interfaces

| API | Description |
| --- | --- |
| [overwolf.windows.mediaplayerelement.CreateResult](./Interfaces/overwolf.windows.mediaplayerelement.CreateResult) | Result of a `create` call, containing the new player's ID. |
| [overwolf.windows.mediaplayerelement.GetProgressResult](./Interfaces/overwolf.windows.mediaplayerelement.GetProgressResult) | Result of a `getProgress` call, containing the current playback position. |
| [overwolf.windows.mediaplayerelement.PlaybackEvent](./Interfaces/overwolf.windows.mediaplayerelement.PlaybackEvent) | Event data for media player playback events. |
| [overwolf.windows.mediaplayerelement.SetVideoResult](./Interfaces/overwolf.windows.mediaplayerelement.SetVideoResult) | Result of a `setVideo` call, containing the video duration. |

## Variables

| API | Description |
| --- | --- |
| [overwolf.windows.mediaplayerelement.onPlaybackEnded](./Variables/overwolf.windows.mediaplayerelement.onPlaybackEnded) | Fired when playback ends. |
| [overwolf.windows.mediaplayerelement.onPlaybackError](./Variables/overwolf.windows.mediaplayerelement.onPlaybackError) | Fired when there was an error while trying to open a video. |
| [overwolf.windows.mediaplayerelement.onPlaybackPaused](./Variables/overwolf.windows.mediaplayerelement.onPlaybackPaused) | Fired when playback is paused. |
| [overwolf.windows.mediaplayerelement.onPlaybackStarted](./Variables/overwolf.windows.mediaplayerelement.onPlaybackStarted) | Fired when playback is starting/resuming. |
| [overwolf.windows.mediaplayerelement.onPlaybackStopped](./Variables/overwolf.windows.mediaplayerelement.onPlaybackStopped) | Fired when playback is stopped. |
| [overwolf.windows.mediaplayerelement.onScreenPropertyChanged](./Variables/overwolf.windows.mediaplayerelement.onScreenPropertyChanged) | Fired when native window (or OSR on desktop) moved to other monitoror when current monitor resolution changed |
