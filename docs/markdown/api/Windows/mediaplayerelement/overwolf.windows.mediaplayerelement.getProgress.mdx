Overwolf APIs / overwolf.benchmarking / overwolf / windows / mediaPlayerElement / getProgress

```ts
function getProgress(
  id: number,
  callback: CallbackFunction<GetProgressResult>,
): void;
```

Gets the current progress, in seconds, of the playback.

## Parameters

| Parameter  | Type                                                                                                                                                      | Description                                                              |
| ---------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------ |
| `id`       | `number`                                                                                                                                                  | The id of the player.                                                    |
| `callback` | `CallbackFunction`\&lt;`GetProgressResult`\&gt; | A callback function which will be called with the status of the request. |

## Returns

`void`
