Overwolf APIs / overwolf / overwolf / windows / MessageReceivedEvent

Event data fired when a window receives a message via `sendMessage`.

## Properties

| Property                                | Type     | Description                                     |
| --------------------------------------- | -------- | ----------------------------------------------- |
| <a id="property-content"></a> `content` | `any`    | The content of the received message.            |
| <a id="property-id"></a> `id`           | `string` | The ID of the window that received the message. |
