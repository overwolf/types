Overwolf APIs / overwolf / overwolf / windows / ODKRect

A rectangle defined by its top-left origin, width, and height, all in pixels.

## Properties

| Property                              | Type     | Description                                     |
| ------------------------------------- | -------- | ----------------------------------------------- |
| <a id="property-height"></a> `height` | `number` | Height of the rectangle in pixels.              |
| <a id="property-left"></a> `left`     | `number` | Distance from the left of the screen in pixels. |
| <a id="property-top"></a> `top`       | `number` | Distance from the top of the screen in pixels.  |
| <a id="property-width"></a> `width`   | `number` | Width of the rectangle in pixels.               |
