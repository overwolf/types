Overwolf APIs / overwolf / overwolf / windows / WindowProperties

Optional properties used when obtaining a declared window.

## Properties

| Property                                                      | Type      | Description                                            |
| ------------------------------------------------------------- | --------- | ------------------------------------------------------ |
| <a id="property-enablepopupblocker"></a> `enablePopupBlocker` | `boolean` | Whether to enable the popup blocker for this window.   |
| <a id="property-nativewindow"></a> `nativeWindow`             | `boolean` | Whether to create the window as a native (CEF) window. |
