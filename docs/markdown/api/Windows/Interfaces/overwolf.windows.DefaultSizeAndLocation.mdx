Overwolf APIs / overwolf / overwolf / windows / DefaultSizeAndLocation

Instructs `obtainDeclaredWindow` to apply the manifest's default size and location.

## Properties

| Property                                                                    | Type      | Description                                                                             |
| --------------------------------------------------------------------------- | --------- | --------------------------------------------------------------------------------------- |
| <a id="property-usedefaultsizeandlocation"></a> `useDefaultSizeAndLocation` | `boolean` | Set to `true` to use the manifest-defined size and position instead of any saved state. |
