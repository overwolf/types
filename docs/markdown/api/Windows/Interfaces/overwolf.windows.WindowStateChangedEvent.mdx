Overwolf APIs / overwolf / overwolf / windows / WindowStateChangedEvent

Event data fired when a window's state changes.

## Properties

| Property                                                                  | Type                                                                  | Description                                         |
| ------------------------------------------------------------------------- | --------------------------------------------------------------------- | --------------------------------------------------- |
| <a id="property-app_id"></a> `app_id`                                     | `string`                                                              | The ID of the app that owns the window.             |
| <a id="property-window_id"></a> `window_id`                               | `string`                                                              | The ID of the window whose state changed.           |
| <a id="property-window_name"></a> `window_name`                           | `string`                                                              | The name of the window as declared in the manifest. |
| <a id="property-window_previous_state"></a> `window_previous_state`       | `string`                                                              | The previous state as a plain string (legacy).      |
| <a id="property-window_previous_state_ex"></a> `window_previous_state_ex` | `WindowStateEx` | The previous state as a typed enum value.           |
| <a id="property-window_state"></a> `window_state`                         | `string`                                                              | The new state as a plain string (legacy).           |
| <a id="property-window_state_ex"></a> `window_state_ex`                   | `WindowStateEx` | The new state as a typed enum value.                |
