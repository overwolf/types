Overwolf APIs / overwolf / overwolf / windows / onScreenPropertyChangedEvent

Event data fired when a window's screen or monitor properties change.

## Properties

| Property                                | Type                                            | Description                              |
| --------------------------------------- | ----------------------------------------------- | ---------------------------------------- |
| <a id="property-id"></a> `id`           | `string`                                        | The ID of the affected window.           |
| <a id="property-monitor"></a> `monitor` | `Display` | The updated display/monitor information. |
| <a id="property-name"></a> `name`       | `string`                                        | The name of the affected window.         |
