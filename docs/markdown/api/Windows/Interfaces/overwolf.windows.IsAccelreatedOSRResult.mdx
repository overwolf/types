Result of `isAccelreatedOSR`, reporting GPU acceleration status for an OSR window.

## Extends

- `WindowIdResult`

## Properties

| Property                                         | Type      | Description                                                                  |
| ------------------------------------------------ | --------- | ---------------------------------------------------------------------------- |
| <a id="property-accelerated"></a> `accelerated?` | `boolean` | Whether the window is using GPU acceleration.                                |
| <a id="property-error"></a> `error?`             | `string`  | Information regarding the error (if an error occurred)                       |
| <a id="property-optimized"></a> `optimized?`     | `boolean` | Whether rendering is optimized (only valid in-game for accelerated windows). |
| <a id="property-success"></a> `success`          | `boolean` | Whether the method executed successfully or not.                             |
| <a id="property-supported"></a> `supported?`     | `boolean` | Whether GPU acceleration is supported on this machine.                       |
| <a id="property-window_id"></a> `window_id?`     | `string`  | The ID of the affected window.                                               |
