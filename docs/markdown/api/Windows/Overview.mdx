# Overview

Use this API to manage Overwolf windows — create, position, resize, show, hide, and control the lifecycle and styling of your app's UI windows.

| API | Description |
| --- | --- |
| [overwolf.windows.enums.FlashBehavior](./enums/overwolf.windows.enums.FlashBehavior) | Controls the taskbar flash behavior for a window. |
| [overwolf.windows.enums.MessagePromptIcon](./enums/overwolf.windows.enums.MessagePromptIcon) | The icon displayed in a message prompt dialog. |
| [overwolf.windows.enums.WindowDragEdge](./enums/overwolf.windows.enums.WindowDragEdge) | The edge or corner from which a window drag-resize operation originates. |
| [overwolf.windows.enums.WindowStateEx](./enums/overwolf.windows.enums.WindowStateEx) | The possible display states of an Overwolf window. |
| [overwolf.windows.enums.WindowStyle](./enums/overwolf.windows.enums.WindowStyle) | Visual and input-handling styles that can be applied to a window. |
| [overwolf.windows.enums.WindowType](./enums/overwolf.windows.enums.WindowType) | The rendering context type of a window. |
| [overwolf.windows.bringToFront](./Functions/overwolf.windows.bringToFront) | Brings the requested window to the front. |
| [overwolf.windows.changePosition](./Functions/overwolf.windows.changePosition) | Changes the window position in pixels from the top left corner. |
| [overwolf.windows.changeSize](./Functions/overwolf.windows.changeSize) | Changes the window size to the new width and height, in pixels. |
| [overwolf.windows.close](./Functions/overwolf.windows.close) | Closes the window. |
| [overwolf.windows.displayMessageBox](./Functions/overwolf.windows.displayMessageBox) | Displays a customized popup message prompt. |
| [overwolf.windows.dragMove](./Functions/overwolf.windows.dragMove) | Start dragging a window. |
| [overwolf.windows.dragResize](./Functions/overwolf.windows.dragResize) | Start resizing the window from a specific edge or corner. |
| [overwolf.windows.flash](./Functions/overwolf.windows.flash) | Flashes a window. |
| [overwolf.windows.getCurrentWindow](./Functions/overwolf.windows.getCurrentWindow) | Calls the given callback function with the current window object as a parameter. |
| [overwolf.windows.getMainWindow](./Functions/overwolf.windows.getMainWindow) | Returns a window object of the index page. |
| [overwolf.windows.getOpenWindows](./Functions/overwolf.windows.getOpenWindows) | Returns an array of all open windows as objects. The objects can be manipulated like any other window. |
| [overwolf.windows.getWindow](./Functions/overwolf.windows.getWindow) | Returns WindowResult object for a specific open window. |
| [overwolf.windows.getWindowDPI](./Functions/overwolf.windows.getWindowDPI) | Get Window DPI. |
| [overwolf.windows.getWindowsStates](./Functions/overwolf.windows.getWindowsStates) | Returns the state of all windows owned by the app (normal/minimized/maximized/closed). |
| [overwolf.windows.getWindowState](./Functions/overwolf.windows.getWindowState) | Returns the state of the window (normal/minimized/maximized/closed). |
| [overwolf.windows.hide](./Functions/overwolf.windows.hide) | Hides the window. |
| [overwolf.windows.isAccelreatedOSR](./Functions/overwolf.windows.isAccelreatedOSR) | For OSR window only (for other window the callback return `error` status. &#123; "status": string (\|error, success\|), "reason": string (error reason), "accelerated": bool, "optimized" : bool (for accelerated windows only and only valid in Game) &#125; |
| [overwolf.windows.isMuted](./Functions/overwolf.windows.isMuted) | Is window muted. |
| [overwolf.windows.isWindowVisibleToUser](./Functions/overwolf.windows.isWindowVisibleToUser) | Is window fully visible to user (has overlap windows) |
| [overwolf.windows.maximize](./Functions/overwolf.windows.maximize) | Maximizes the window. |
| [overwolf.windows.minimize](./Functions/overwolf.windows.minimize) | Minimizes the window. |
| [overwolf.windows.muteAll](./Functions/overwolf.windows.muteAll) | Mute all sound source include all excluded white list |
| [overwolf.windows.obtainDeclaredWindow](./Functions/overwolf.windows.obtainDeclaredWindow) | Creates or returns a window by the window name that was declared in the manifest. |
| [overwolf.windows.openOptionsPage](./Functions/overwolf.windows.openOptionsPage) | Opens the options page specified in the manifest file. Does nothing if no such page has been specified. |
| [overwolf.windows.removeWindowStyle](./Functions/overwolf.windows.removeWindowStyle) | Remove Window In Game Styling |
| [overwolf.windows.restore](./Functions/overwolf.windows.restore) | Restores a minimized window. |
| [overwolf.windows.sendMessage](./Functions/overwolf.windows.sendMessage) | Sends a message to an open window. |
| [overwolf.windows.sendToBack](./Functions/overwolf.windows.sendToBack) | Sends the window to the back. |
| [overwolf.windows.setDesktopOnly](./Functions/overwolf.windows.setDesktopOnly) | Sets whether the window should be injected to games or not. |
| [overwolf.windows.setMinSize](./Functions/overwolf.windows.setMinSize) | Changes the window minimum size to the new width and height, in pixels. |
| [overwolf.windows.setMute](./Functions/overwolf.windows.setMute) | Set current window Mute state. |
| [overwolf.windows.setPosition](./Functions/overwolf.windows.setPosition) | Change window position (see SetWindowPositionProperties)) |
| [overwolf.windows.setRestoreAnimationsEnabled](./Functions/overwolf.windows.setRestoreAnimationsEnabled) | Sets whether the window should have minimize/restore animations while in game. |
| [overwolf.windows.setTopmost](./Functions/overwolf.windows.setTopmost) | Change the window's topmost status. Handle with care as topmost windows can negatively impact user experience. |
| [overwolf.windows.setWindowStyle](./Functions/overwolf.windows.setWindowStyle) | Add Window In Game styling |
| [overwolf.windows.setZoom](./Functions/overwolf.windows.setZoom) | Set window zoom level (0.0 for reset). |
| [overwolf.windows.AltF4BlockedEvent](./Interfaces/overwolf.windows.AltF4BlockedEvent) | Event data fired when the user attempts to close a window with Alt+F4 and is blocked. |
| [overwolf.windows.ChangeWindowSizeParams](./Interfaces/overwolf.windows.ChangeWindowSizeParams) | Parameters for the `changeSize` overload that supports DPI-aware resizing. |
| [overwolf.windows.DefaultSizeAndLocation](./Interfaces/overwolf.windows.DefaultSizeAndLocation) | Instructs `obtainDeclaredWindow` to apply the manifest's default size and location. |
| [overwolf.windows.DragMovedResult](./Interfaces/overwolf.windows.DragMovedResult) | Result of a `dragMove` operation, reporting how far the window moved. |
| [overwolf.windows.DragResizeResult](./Interfaces/overwolf.windows.DragResizeResult) | Result of a drag-resize operation, containing the new dimensions. |
| [overwolf.windows.GetWindowsStatesResult](./Interfaces/overwolf.windows.GetWindowsStatesResult) | Result of `getWindowsStates`, describing the state of all app windows. |
| [overwolf.windows.GetWindowStateResult](./Interfaces/overwolf.windows.GetWindowStateResult) | Result of `getWindowState`, describing the current state of a single window. |
| [overwolf.windows.IsAccelreatedOSRResult](./Interfaces/overwolf.windows.IsAccelreatedOSRResult) | Result of `isAccelreatedOSR`, reporting GPU acceleration status for an OSR window. |
| [overwolf.windows.IsMutedResult](./Interfaces/overwolf.windows.IsMutedResult) | Result of `isMuted`, indicating whether the window's audio is muted. |
| [overwolf.windows.IsolatedIframeProcessCrashedEvent](./Interfaces/overwolf.windows.IsolatedIframeProcessCrashedEvent) | Event data fired when an isolated iframe process crashes. |
| [overwolf.windows.IsWindowVisibleToUserResult](./Interfaces/overwolf.windows.IsWindowVisibleToUserResult) | Result of `isWindowVisibleToUser`, indicating how much of the window is visible. |
| [overwolf.windows.MessageBoxParams](./Interfaces/overwolf.windows.MessageBoxParams) | Parameters for the `displayMessageBox` prompt dialog. |
| [overwolf.windows.MessageReceivedEvent](./Interfaces/overwolf.windows.MessageReceivedEvent) | Event data fired when a window receives a message via `sendMessage`. |
| [overwolf.windows.ODKRect](./Interfaces/overwolf.windows.ODKRect) | A rectangle defined by its top-left origin, width, and height, all in pixels. |
| [overwolf.windows.onScreenPropertyChangedEvent](./Interfaces/overwolf.windows.onScreenPropertyChangedEvent) | Event data fired when a window's screen or monitor properties change. |
| [overwolf.windows.SetWindowPositionProperties](./Interfaces/overwolf.windows.SetWindowPositionProperties) | Describes a relative position for a window with respect to another process window. |
| [overwolf.windows.WindowIdResult](./Interfaces/overwolf.windows.WindowIdResult) | Result of window operations that return only the window's ID. |
| [overwolf.windows.WindowInfo](./Interfaces/overwolf.windows.WindowInfo) | Describes a currently known Overwolf window and its current state. |
| [overwolf.windows.WindowProperties](./Interfaces/overwolf.windows.WindowProperties) | Optional properties used when obtaining a declared window. |
| [overwolf.windows.WindowResult](./Interfaces/overwolf.windows.WindowResult) | Result of window operations that return a `WindowInfo` object. |
| [overwolf.windows.WindowStateChangedEvent](./Interfaces/overwolf.windows.WindowStateChangedEvent) | Event data fired when a window's state changes. |
| [mediaplayerelement](./mediaplayerelement/Overview) | Use this API to embed and control native media player elements within Overwolf windows for video playback. |
| [overwolf.windows.onAltF4Blocked](./Variables/overwolf.windows.onAltF4Blocked) | Fired when the user was prevented from closing a window using Alt+F4 |
| [overwolf.windows.onIsolatedIframeProcessCrashed](./Variables/overwolf.windows.onIsolatedIframeProcessCrashed) | Fired when out of process iframe crashed. |
| [overwolf.windows.onMainWindowRestored](./Variables/overwolf.windows.onMainWindowRestored) | Fired when the main window is restored. |
| [overwolf.windows.onMessageReceived](./Variables/overwolf.windows.onMessageReceived) | Fired when this window received a message. |
| [overwolf.windows.onStateChanged](./Variables/overwolf.windows.onStateChanged) | Fired when the state of a window is changed. |
| [overwolf.windows.onSubscriptionChanged](./Variables/overwolf.windows.onSubscriptionChanged) | Fired when current extension subscription has changed. |
