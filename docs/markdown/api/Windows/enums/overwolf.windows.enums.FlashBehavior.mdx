Overwolf APIs / overwolf / overwolf / windows / enums / FlashBehavior

Controls the taskbar flash behavior for a window.

## Enumeration Members

| Enumeration Member                                    | Value         | Description                                               |
| ----------------------------------------------------- | ------------- | --------------------------------------------------------- |
| <a id="enumeration-member-automatic"></a> `automatic` | `"automatic"` | Flash behavior is determined automatically by the system. |
| <a id="enumeration-member-off"></a> `off`             | `"off"`       | The window does not flash.                                |
| <a id="enumeration-member-on"></a> `on`               | `"on"`        | The window flashes continuously.                          |
