Overwolf APIs / overwolf / overwolf / windows / enums / WindowStateEx

The possible display states of an Overwolf window.

## Enumeration Members

| Enumeration Member                                    | Value         | Description                                |
| ----------------------------------------------------- | ------------- | ------------------------------------------ |
| <a id="enumeration-member-closed"></a> `closed`       | `"closed"`    | The window is closed and no longer exists. |
| <a id="enumeration-member-hidden"></a> `hidden`       | `"hidden"`    | The window exists but is not visible.      |
| <a id="enumeration-member-maximized"></a> `maximized` | `"maximized"` | The window occupies the full screen.       |
| <a id="enumeration-member-minimized"></a> `minimized` | `"minimized"` | The window is minimized to the taskbar.    |
| <a id="enumeration-member-normal"></a> `normal`       | `"normal"`    | The window is visible at its normal size.  |
