Overwolf APIs / overwolf / overwolf / windows / setZoom

```ts
function setZoom(winzoomFactorowId: number, windowId: string): void;
```

Set window zoom level (0.0 for reset).

## Parameters

| Parameter           | Type     | Description                              |
| ------------------- | -------- | ---------------------------------------- |
| `winzoomFactorowId` | `number` | The zoome factor.                        |
| `windowId`          | `string` | The window id, empty for current window. |

## Returns

`void`
