function setMute(mute: boolean, callback: CallbackFunction<Result>): void;

## Parameters

| Parameter  | Type                                                                                                                                | Description                            |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------- |
| `mute`     | `boolean`                                                                                                                           | window mute (on\off).                  |
| `callback` | `CallbackFunction`\<`Result`\> | Called with the result of the request. |

## Returns

`void`
