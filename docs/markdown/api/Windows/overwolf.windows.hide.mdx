Overwolf APIs / overwolf.benchmarking / overwolf / windows / hide

```ts
function hide(
  windowId: string,
  callback?: CallbackFunction<WindowIdResult>,
): void;
```

Hides the window.

## Parameters

| Parameter   | Type                                                                                                                                          | Description                           |
| ----------- | --------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------- |
| `windowId`  | `string`                                                                                                                                      | The id or name of the window to hide. |
| `callback?` | `CallbackFunction`\&lt;`WindowIdResult`\&gt; | Called after the window is hidden.    |

## Returns

`void`
